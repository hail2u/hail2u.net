HTTP status messages

Last Update: 2004-09-03T04:20:40+09:00

サーバーから返されるHTTPステータス番号のリストです。

1xx: Information

100 Continue
リクエストの一部は、サーバーに受け取られましたが、それが拒絶されていない限り、クライアントはリクエストを継続するべきです。
101 Switching Protocols
サーバーはプロトコルを変更しました。

2xx: Successful

200 OK
リクエストは成功しました。
201 Created
リクエストは完了し、新しいリソースが作成されました。
202 Accepted
リクエストはサーバーに受け取られましたが、処理は完了していません。
203 Non-authoritative Information
?
204 No Content
リクエストは完了しましたが、本文はありません。
205 Reset Content
?
206 Partial Content
部分的 GET を受け入れました。

3xx: Redirection

300 Multiple Choices
最大 5 つのアドレスのリンク先に、リンクを選択することによってジャンプすることが出来ます。
301 Moved Permanently
リクエストされたページは新しい URL に移動しました。
302 Found
リクエストされたページは新しい URL に一時的に移動しました。
303 See Other
リクエストされたページは異なる URL にあります。
304 Not Modified
リクエストされたページは更新されていません。
305 Use Proxy
プロクシを使用してアクセスしてください。
306 Unused
このコードはもはや使用されていませんが、将来のために予約されています。
307 Temporary Redirect
リクエストされたページは新しい URL に一時的に移動しました。

4xx: Client Error

400 Bad Request
このリクエストはサーバーに理解できません。
401 Unauthorized
リクエストされたページはユーザー名とパスワードを入力する必要があります。
402 Payment Required
このコードはまだ使用することが出来ません。
403 Forbidden
リクエストされたページはアクセスが禁止されています。
404 Not Found
リクエストされたページは見つかりません。
405 Method Not Allowed
リクエストで使用されている方法は許可されていません。
406 Not Acceptable
サーバーは、クライアントに受けとれないレスポンスを作成することができます。
407 Proxy Authentication Required
このリクエストを送信できるようにプロクシーサーバを設定しなければなりません。
408 Request Timeout
リクエストはタイムアウトしました。
409 Conflict
リクエストは、矛盾しているために完了できませんでした。
410 Gone
リクエストされたページは存在しません。
411 Length Required
「Content-Legth」が定義されていません。サーバーは、それのないリクエストを受け入れません。
412 Precondition Failed
リクエストの必須条件が足りません。
413 Request Entity Too Large
リクエストの実態が大きすぎるので、サーバーはリクエストを受諾しません。
414 Request-url Too Long
URL が長すぎます。
415 Unsupported Media Type
このメディアタイプはサポートされていません。
416
?
417 Expectation Failed
?

5xx: Server Error

500 Internal Server Error
サーバーは予期せぬエラーに見舞われました。
501 Not Implemented
サーバーは要求された機能を持っていません。
502 Bad Gateway
サーバーは無効な応答を受け取りました。
503 Service Unavailable
サーバーは一時的に過負荷がかかっているか、ダウンしています。
504 Gateway Timeout
サーバーのゲートウェイでタイムアウトしました。
505 HTTP Version Not Supported
サーバーはこの「http protocol」バージョンをサポートしていません。
