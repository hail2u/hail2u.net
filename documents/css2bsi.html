CSS 2 browser support information

←

各ブラウザのCSS 2.0で定義されているプロパティなどのサポート状況の表です。

略称の意味

  • WIE4 - Windows Internet Explorer 4
  • WIE5 - Windows Internet Explorer 5/5.01
  • WIE5.5 - Windows Internet Explorer 5.5
  • WIE6 - Windows Internet Explorer 6(strictモード)
  • N4 - Netscape 4
  • M - Mozilla 1.x(strictモード) または Netscape 7(strictモード)
  • O6 - Opera 6.x
  • MIE5 - Macintosh Internet Explorer 5(strictモード)

記号の意味

  • ○ - サポートしている。または、多少のバグがあるがサポートしている。
  • △ - 普通に使うにはちょっとつらいバグがあるが、サポートしている。
  • × - サポートしていない。または、使い物にならないバグがあるが、サポートしている。
