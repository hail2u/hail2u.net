WSHでNowPlaying

ということでWSHで、Winampの再生中のファイルのID3タグを無理やり読んでみる。WSHでっつーのにはかなり語弊がありますが。。。とりあえずダイアログ出すところまでは出来たり。

必要なもの。

  • Windows Script 5.6
  • Winamp
  • Winamp Com
  • Tag

WSHでっつーのには語弊があるどころの騒ぎじゃない。

続きは作るかわかりません。

再生中のファイル名さえ取得できればいいので、Winamp COMじゃなくても良いかも。Winamp COM好きなんで。ちなみに、Winamp COMがあると、

var objWinamp = new ActiveXObject("WinampCOM.Application");
objWinamp.Play();
objWinamp = null;
WScript.Quit();

てなスクリプトで再生とか、

var objWinamp = new ActiveXObject("WinampCOM.Application");
objWinamp.Stop();
objWinamp = null;
WScript.Quit();

てなスクリプトで停止とか、

var objWinamp = new ActiveXObject("WinampCOM.Application");
var sTitle = objWinamp.CurrentSongTitle;
objWinamp = null;
WScript.Echo(sTitle);
WScript.Quit();

てなスクリプトで再生中のファイルのタイトルなんかをダイアログで表示とか出来たりします。
