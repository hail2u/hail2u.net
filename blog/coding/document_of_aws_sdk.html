Amazon Web Services SDKのドキュメント

ブラウザで開く度に「ガガガッ」というけたたましい音を鳴らしてくれるAmazon Web Services SDKのドキュメント。なんでフロッピー・ディスクにアクセスしにいくのだろうと思いつつ、たまにしか読まないので放置していたのをとりあえずチェックしてみたら、link要素のhref属性の値がfile:///A:/...とかになってたよ。なんじゃそりゃ。

とりあえず、そのhref属性の値を、

surfmapjs/bookstyle/bookstyle.css

とすると意図した表示になるようですね。

文字ちっさ！

HTMLヘルプ作ろうかな・・・とか思ったけど、エディタから行き来とかするわけでもないし、ブラウザで見れるほうが圧倒的に良いな。CSSをいじろう。
