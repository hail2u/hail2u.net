Pure JavaScript Wiki Like Formatter #3

なんとなくPure JavaScript Wiki Like Formatterの続きを作ってみる。textareaの改行コードがMacだと\rだったりしそうな気がするので、とりあえず改行コードを\nに統一してから\nで分割にしてみました。他にプレビュー機能をつけてみたりとか。作業時間は3分。

改行コードの統一はかなりテキトウなので、Perlみたいに環境によって\rや\nが示すものが違うようなことがあったりするとダメダメっぽい。ちゃんと考えないと。そもそも各環境の各ブラウザでのtextareaでの改行コードをちゃんと調べないとならない説。

プレビューはイマイチ何がなにやらわからなそうな感じなので付けてみました。

画像貼り付けとかリストのネストとか作りたいですが、めんどくさくて死ねそう(ネストは特に)。
