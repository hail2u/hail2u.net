スタイルシートを無効にするJavaScript

<script type="text/javascript" defer>
	var pWin = external.menuArguments;
	var docs = pWin.document;
	for(i = 0; i < docs.styleSheets.length; i++){
		docs.styleSheets.item(i).disabled = true;
	}
</script>

Internet Explorerの右クリックメニューに仕込んで、いろんなサイト見て回ると楽しい。
