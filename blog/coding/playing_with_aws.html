Amazon Web Servicesで遊ぶ

←

いまさらAmazon Web Servicesで遊んでます。とりあえずサブ・ドメイン、aws.hail2u.netで運営。ま、単に自分でAmazon.comやAmazon.co.jpで検索するのがかったるいとか言う理由なわけで。大きな声では言いませんが、検索結果のリンクにはアソシエイト・タグが仕込まれてます。

やはり単なる検索は作る分には楽しいですが、使っても面白くないですね。まけん作ま、検索が面白くても困るわけなんですが。とりあえずこれは自分で使うので残しておいて、何か面白いアイディア考えついたら追加してみようかと・・・思うだけ思っておく。

ASINによる検索は作ろう。
