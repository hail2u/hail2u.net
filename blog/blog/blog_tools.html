Blog Tools

WebAttack.comにBlog Toolsというカテゴリが出来ていました。それだけ。目新しいものは特に無し。

  • Shareware downloads Web Authoring - Blog Tools
  • Freeware downloads Web Authoring - Blog Tools

こういういわゆるBlog Toolとはまた別に、最近は、RSS Aggregatorにblogのエントリーをポストする機能とかつけるのが流行り始めてるみたい。個人的にはブラウザで動作するHTML編集アプリケーション（Textileとか）やWikiのような編集機能を組み込んだblogと、BookmarkletやIEのコンテキストメニュー拡張なんかを組み合わせて使う方が良いような気がするんですが。
