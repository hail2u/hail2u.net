Statlogger

Statloggerは特定の形式で件名を記述しstat@statlogger.comへメールを送ると、メールでグラフの画像を返してくれるだけのウェブサービス。アプリ不要、登録不要、ログイン不要でとにかくメールを送るだけ。

例えば上記スクリーンショットのような折れ線グラフを作るには以下のような件名でメールを送る。

Test +1; Test +10; Test -3; Test -2; Test +1; Test -5; Test +2

本文は空で良い。そうするとこの場合は7件のメールが返ってきて、最後のメールでスクリーンショットのようなグラフになる。数字の前がグラフ名になり、それに続けた数字がグラフの値になる。数字に+や-を付けると相対変化の折れ線グラフになり、付けないと数字の単純な変化グラフになる。これらの数字は例のようにセミコロンで区切ってまとめて渡すことも、一度にひとつだけ送ることもできる。どちらの場合も後に同じグラフ名を使って数字を送れば、そのグラフの続きに追加することができる。

メールアドレスの扱い辺りにすごく不安が残るけど、手軽で面白そう。公開しないようなパーソナルなデータの記録に向いてるのはもちろん、返信先をメーリングリストにしてグラフを公開するみたいなこともできそう。
