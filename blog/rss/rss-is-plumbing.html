RSS is plumbing.

RSSはいろんな勝ち組ウェブサービスにほぼ瀕死に追い込まれてて、影響は意外なところにも波及する。Twitter API v1の終了により、自分のツイートを自由に自分のウェブサイトに載せることがすごく面倒になったとかもそのひとつ。自分のなのにね。スクレイピングの時代に戻ったりして。

お前のものは俺のもの。俺のものは俺のもの。

後ろで定義したセレクターも@extendで参照できる
