Brand-new Bloglines

方々で話題になっていますが、Bloglinesが全面リニューアルされました。

以下、気づいたとこ。

  • 過去ログの表示が速くなった気がする
  • Share Subscriptionsが新機能Blogの一部の機能になったっぽい
  • 全体的に詰まりすぎ（文字がね）
  • 文字サイズ変えられね
  • Edit（旧 Manage Subscriptions）からCheck RangesとかSort Decsendingとか消えた？
  • RSSの追加わかりづら（右上のセレクトボックスの一番下）

と、なんだかマイナス点ばかり目に付いてしまいました。

新機能のBlogとClipですが、Blogはpublic/privateの2択、Clippingはprivate限定というわけで、公開しても良いメモはBlogへ、エログロ公開しちゃうと品性が疑われるものはClipへという感じになるのかな。BlogとClipそれぞれに登録した記事は、その記事を開いて右上のeditから移動できたりするみたい。また、Blogはある記事へのリンク+自分のコメント、Clipはある記事そのまんまという感じみたいですね。

あんまり関係ないですけど、ユーザー・スタイルシートに、

td.article pre {
  width:auto;
  overflow:scroll;
}

と書いておくと、記事にpre要素が含まれていて、そのせいで横スクロールバーが出るのを回避できるみたいです。tdにclass名振られたのは今回のリニューアルからかな？IEだとwidthの挙動がアレなので、

td.article pre {
  width:90%;
  overflow-x:scroll;
}

とかかもしれない（試す気無し）。

RssMemoの存在意義がさらに薄れてきたなぁ・・・。まぁ使うけど。

Sortとかは、今までのように物理的にSortするわけではなく、フォルダごとの設定として保持するみたいですね。とりあえずアルファベット順に並べておきたい人には便利といえば便利（My Feeds→Sortをクリック→Check Allクリック→By Name: A-Z→Sortボタンを押すで全部アルファベット順に設定できました）。だけど、直感的ではないかもしれない。Check Rangesが消えた理由はここら辺にある模様。SortにはBy Number Unread（未読数）とBy Date: Oldest First/By Date: Newest First（最新記事の日付け昇順・降順）というのも追加されてましたね。フォルダ分けしない人に向いてるかもしれない。

あまりに色が多くて見づらいというか違和感があるので、ユーザー・スタイルシートでいろいろスタイルを取っ払ってみました。

すっきり。

ユーザー・スタイルシートに書いたのは以下のような感じ。

div.content-main .channel {
  border-bottom:none !important;
  color:inherit !important;
  background-color:#1393c0 !important;
}

div.content-main .channel * {
  color:#fff !important;
  background-color:transparent !important;
}

div.content-main .channel img {
  padding:0 !important;
  display:none !important;
}

div.content-main ul.channel_nav {
  color:inherit !important;
  background-color:inherit !important;
}

div.content-main ul.channel_nav li {
  border-right:none !important;
}

div.content-main ul.channel_nav li a {
  color:inherit !important;
  background-color:inherit !important;
}

div.content-main .item_nav ul {
  border-top:none !important;
  border-bottom:none !important;
}

div.content-main .header_nav ul {
  border-bottom:none !important;
  background-color:inherit !important;
}

div.content-main .item td.article {
  line-height:1.75 !important;
}

div.content-main .item td.article pre {
  width:auto !important;
  overflow:auto !important;
}

確認不足なので、不具合起こるかもしれませんが、そこはまぁ勘で修正してください。セレクタの先頭にdiv.content-mainを付けてるので、他のサイトで誤爆するってことはまずないと思います。
