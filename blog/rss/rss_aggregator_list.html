RSS Aggregatorリスト

とりあえずチェックしたクライアント側で動作するRSS Aggregatorsだけ、まとめようかなと。MacやLinuxでWebを徘徊することがほとんど無いので、Windowsで使えるものばかりになっているところは勘弁してください。

アプリケーション名の前の数字は、機能面の5段階評価での採点です。かなり適当ですので参考までに。評価基準は以下のとおり。

  • 一般的な気がするインターフェイスを備える
  • RSSファイルの追加・削除がしやすい
  • 環境(RSSファイル提供元/アプリケーション実行環境)に優しい
  • UTF-8のRSSファイルが読め、表示が出来る(バイナリ/ソースをほんの少し書き換えれば済むものも可)
  • 個人的な主観によるその他機能の評価

ということで、事実上4点満点。1点しか取れなかったものは削除してます。また安定しているかとかは、考慮してないです。まだまだ若いカテゴリのアプリケーションだと思うので、そこらへんは大目に見てもいいような気がするので。

Last update at 2004/02/12 (Thu) 10:00

  • 日本のもの:
    • 5 glucose Windows Free
    • なじみやすいメーラーのような3ペイン型のGUI。「センサー」でRSSファイルの提供されていないサイトも同じように読める。
    • 4 Headline-Reader Windows ¥1500
    • 3ペイン+αなGUI。メニューがわかりづらい。とりあえずの機能は備えるが、微妙。
    • 5 HepCat Windows Free
    • 3ペイン+αなGUI。後発だけあり良く練られている感じ。ツリー部分が多少わかりにくいかも・・・とか無理やりケチをつけてみる。良いです。
    • 5 NewsGlue .Net ¥3000 (thanks for liki)
    • なじみやすいメーラーのような3ペイン型のGUI。リマインダや取得した記事をメールで転送など多機能。基本的には無料(らしい)。
    • 3 RSS Reader Panel Mozilla Free
    • XULアプリケーション。ブラウザでRSSを読むことが出来る程度のもの。軽さやブラウザとの連携など見るべきところは多いが、やや機能に欠けるかも。
    • 5 RSSGate XSL(?) Free
    • ローカルサーバーとして動作する。閲覧はブラウザで、整形にはXSLを利用している模様。通常のウェブページもスクリプトを作成すればRSSにコンバートして閲覧できるなど、高機能。
    • 5 RThReader Windows Free
    • 4ペイン型のGUI。挙動は俊敏。多少画面がごちゃごちゃしすぎている嫌いはあるが、整理できるので問題無し。
  • 海外のもの
    • 3 ActiveRefresh Windows $23.95
    • なじみやすいメーラーのような3ペイン型のGUIとエクスプローラ型の2ペインのGUIを選択できる。使いやすい。ライセンス形態がわかりづらく、手を出しにくい。
    • 4 Abilon Windows Free
    • なじみやすいメーラーのような3ペイン型のGUIとエクスプローラ型の2ペインのGUIを選択できる。使いやすい。上記のActiveRefreshのフリー版という位置付け？　驚くほど軽い。日本語通りません、まるで(UTF-8もということ)。すばらしい軽さなのに。
    • 4 Aggie .Net Free(MITL)
    • ローカルサーバーとして動作する。出力はHTMLファイルで自在にカスタマイズできる。RSSファイルの登録がOPMLを私用する場合以外はいまいち。デフォルトの出力HTMLのJavaScriptが挙動不審。
    • 3 Aggreg8 Mozilla Free
    • 独自(単体XULアプリケーションとして)でもMozillaに統合した形でも運用可能。設定が煩雑。ロゴが邪魔。
    • 5 Amphetadesk Perl Free
    • ローカルサーバーとして動作する。出力はHTMLファイルで自在にカスタマイズできる。カテゴリごとに分けるなどは出来ない。
    • 3 Awasu Windows Free
    • なじみやすいメーラーのような3ペイン型のGUI。プラグインで拡張可能。欠点がなさそうに見えるのですが、あらゆる点で微妙に使いづらさを感じる。
    • 3 Beaver .Net Free
    • なじみやすいメーラーのような3ペイン型のGUI。カテゴリ分けが出来ない。ウェブサイトがしょっちゅう止まる。
    • 3 BlogExpress .Net Free
    • 2ペイン型のGUI+タブ切り替えウェブ・ブラウザ内蔵。シンプルにまとまっている。ウェブ・ブラウザ内蔵は微妙。ウェブ・ブラウザに内蔵の方がマシな気が。
    • 4 BottomFeeder Various Free
    • なじみやすいメーラーのような3ペイン型のGUI。プラグインで拡張可能。Smalltalkは好き嫌いが分かれる。
    • 2 Composite .Net Free
    • すっきり一覧表示。低機能。RSSをつらつらと眺めるだけの人には向いてる。
    • 4 clevercactus Java Free
    • RSSファイルも読めるメーラー。PIM的な位置付けか。一般的なメーラーであるOutlook ExpressのUIを踏襲しているので非常に把握しやすい。正直、見損なってました。Plain Textモードでは文字化けしますが、HTMLモードならOKです。
    • 3 Desktop Sidebar Windows Free
    • デスクトップの端に常駐して、RSSの内容を展開するティッカー型の変種。変わり種で面白くはあるが実用には向かないかもしれない。流れの速いニュース・サイトのRSSなどを流すのには強いという面はある。
    • 3 EffNews Windows Free
    • Python製のeffbot経由で使うもの。機能がほとんど無い。でも、面白い。
    • 5 FeedDemon Windows $29.95
    • なじみやすいメーラーのような3ペイン型のGUI。多機能で多少わかりづらい。Beta取れたらシェアウェアになりそう。
    • 3 FeedExpress .Net Free(LGPL)
    • なじみやすいメーラーのような3ペイン型のGUI。特筆するところはなくベーシック。
    • 5 FeedReader Windows Free(GPL)
    • なじみやすいメーラーのような3ペイン型のGUI。ユーザー数多い(と思う)。最近、CVSがあまり更新されてない。
    • 4 FeedWinder Java Free(GPL)
    • なじみやすいメーラーのような3ペイン型のGUI。NetNewsWireそっくり。
    • 3 FetchRss Perl Free(GPL)
    • RSSファイルを巡回して新着をメールで送るPerlスクリプト。単純明快。
    • 2 fetchrss Java Free(LGPL)
    • RSSファイルを巡回して新着をメールで送るJavaアプリケーション。使い方を理解できる人は少ないかもしれない情報の少なさ。実用性は高い。
    • 3 Genecast Windows $18/6 months
    • RSSファイルをニュースグループにして公開してくれるサービス。目の付け所は素晴らしい。高いっす。
    • 4 Headline Viewer Windows Free
    • 2ペイン+αなGUI。設定が激しくわかりづらい。機能的には標準を超えるものがある。
    • 3 HotSheet Java Free
    • 1ペイン型のGUI。シンプル。利点が見えてこない。
    • 3 IE RSS Aggregator Windows Free
    • IEのサイドバーに統合された形で利用。IEを普通に使ってない人には無用(逆にIEを使ってる人には有用)。
    • 4 InfoSnorkel Windows $39.95
    • RSSファイルを巡回してHTMLファイルを出力するアプリケーション。出力はかなり柔軟。
    • 4 K.R.S.S. Linuxx(KDE) Free(GPL)
    • 2ペイン型のGUI。RSSファイルの一覧が独創的で、慣れないと辛い。出力がCSSで自在にカスタマイズできる。
    • 4 KlipFolio Windows Free
    • 漫然と読むには最適なティッカー型。ユーザー数多い(と思う)。
    • 5 Liferea Linux(Gtk) Free(GPL)
    • GtkのFeedReaderクローン。日本語完全対応。ひそかに一押し。
    • 5 NetNewsWire Mac OS X $39.95
    • なじみやすいメーラーのような3ペイン型のGUI。ユーザー数多い(と思う)。欠点がほとんど見つからない。
    • 5 NetNewsWire Lite Mac OS X Free
    • NetNewsWireの機能限定Free版。日付け表示が無いのは若干悲しい。
    • 4 NewzCrawler Windows $24.95
    • RSSファイルを読むことに加え、ブラウザ・nntpクライアントの機能もあり。WYSIWYGなHTMLエディターまで・・・。メールで新着通知の機能も。なんでもあり。
    • 3 News Addict Windows Free
    • 1つのRSSファイルをHTMLに変換して表示と言うものなので、ちょっと扱いづらい。更新されそうも無い。
    • 4 NewsGator Windows $29
    • RSSファイルの新着記事をOutlookで受信。Outlookを使わない人に無用(逆にOutlookを使う人には有用)。これのせいでOutlook落ちたとかシャレになりません。でも、PIMシステム周りのツールとしては出来が良い。
    • 4 NewsMonster Mozilla Free
    • XULアプリケーション。機能の洗練度・設定などどれをとっても一級品。お、重い。
    • 3 nntp//rss Java Free(GPL)
    • RSSファイルをニュースグループに変換してくれるローカルサーバ。nntpクライアントで読めるのはそれなりにうれしい。設定わかりづらい。
    • 4 Pears Python Free
    • wxPythonなPythonアプリケーション。リッチテキストでRSSファイルの各アイテムを表示。低機能だが、未来に期待できる。
    • 3 PopHeadlines .Net Free

    • 4 Syndigator Linux(Perl&Gtk) Free(GPL)
    • なじみやすいメーラーのような3ペイン型のGUI。一通りの機能は揃っており、扱いやすい。
    • 4 Wildgrape NewsDesk .Net Free
    • なじみやすいメーラーのような3ペイン型のGUI。設定簡単。軽い。
