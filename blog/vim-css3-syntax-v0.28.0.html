vim-css3-syntax v0.28.0

vim-css3-syntaxのv0.28.0をリリースした。特に大きな変更はないが、何に使うのかよくわからない(わかっていない)CSS Layout API Level 1が追加されたので、マイナー番号が更新されている。

他の更新にも見るべきところはあまりない。グリッド間の隙間を専用のar単位(aspect ratioの略)で指定するようになり、読みやすくなりそうというくらいだ。実装もない上に、いずれ別の仕様に移すと書いてあるので、気にする必要はないだろう。

同時にVim 8.1への対応を進めている。vim81ブランチを見ればわかるように、ほぼ完了している。削除だけにしたので、ミスはないと思う。ついでにテストやファイル名を整理したりもした。これからVimを更新して、自分で使いつつ、半月くらいは寝かせる予定だ。何か見つけた場合はプル・リクエストの方へ書いて欲しい。

このウェブサイトはながしまきょうがCC-BYの下で作成、公開、そして更新しています。更新情報はRSS (記事だけのRSS)またはXで購読できます。その他、わからないことはもう少し詳しい情報を参照してください。
