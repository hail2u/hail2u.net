page-break-*の話

Oct 26, 2011 · Web Design

印刷向けスタイルシートを書いていて、page-break-insideプロパティをpre要素やtable要素で使ったんですが、どうやらInternet ExplorerとOpera以外は対応してなかったり中途半端みたいですね(2011年10月現在)。

page-break-beforeとpage-break-afterプロパティではalwaysしか使えなかったり、page-break-insideプロパティは全然な感じです。

@media print {
  h1, h2, h3, h4, h5, h6 {
    page-break-after: avoid;
  }

  blockquote, pre, table {
    page-break-inside: avoid;
  }
}

見出しの直後での改ページや引用や整形済みテキスト、表内での改ページを制御したかったんですが……。
