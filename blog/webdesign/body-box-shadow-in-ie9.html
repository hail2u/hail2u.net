IE9でbody要素へbox-shadowが付けられない

タイトルまま。他のブラウザでは大体意図した通りにbox-shadowが付く。box-shadowプロパティーはIE9でもtable要素絡みあたり以外ならまともに動くので、エッジケースと考えて良さそう。一応解決方法もある。

Demo: Inset box-shadow to body in IE9

insetキーワードを使って内側にしたのはわかりやすくするため。あってもなくても結果は変わらない。開発者ツールで見るとプロパティー自体は認識されている。とりあえず再現率は100%。

html {
  background-color: white;
}

などとしてhtml要素の背景色をtransparent以外にすると解決する。でもこれやるとbody要素の背景画像の開始位置がずれることは頭に入れておいた方が良い。borderプロパティーを使ったりすれば何とか出来るのだけど……。
