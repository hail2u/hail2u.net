Calibri LightをCSSで

Jun 26, 2013 · Web Design

Windows 8には最初からCalibri Lightがあり、Windows 7とWindows 2008 Server R2にも更新プログラムで来ている。イタリックもちゃんとある。が、CSSから手軽に切り替えて使えるわけではないようだというメモ。

View Demo: Use Calibri Light in CSS

Calibriファミリーなようなので、font-weightプロパティーでいけるかと思ったらInternet Explorer 10 (と9)くらいしかファミリーと認識してくれない。ので直接Calibri Lightを指定する必要がある。

もうひとつ気になるのはレンダリング。Chrome 27では通常のCalibriよりも場所をとるようで一行に入る文字数が減る。Firefox 21では通常のCalibriとほぼ同じ。Internet Explorer 10では若干コンパクト。

とにかくCalibriの綴りは覚えた。
