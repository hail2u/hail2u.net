CSSのお勉強

div {
  background-position:12px center; /* not valid */
  background-position:12px 50%; /* valid */
}

キーワードと数値は組み合わせてはいけないらしい。IEのゆるゆる解釈で通ったから、あんまり気にしてなかった。またひとつ、おりこうになりました。

追記

CSS 2.1ではOKになっているので、もうvalidで多くのブラウザで問題なく通ります。
