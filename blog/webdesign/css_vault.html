CSS Vault

CSSでデザインされたサイトを紹介するgalleryとCSSデザインのテクニックを公開するresourcesの2つのセクションからなる、CSS Vaultというサイトを、hebig.org/blogのQuick Links, November 13経由で知りました。

galleryもresourcesもリンクを張るという形で集積したものっぽいので、RSSで読むのが良いかと思います。情報収集の場としてはなかなか良いとは思うのですが、他人の褌で相撲をとっている感が強く、ちょっとどうかな・・・と思っちゃいました。

ピックアップしている内容自体はかなりのものだとは思います。
