CSS Depot

CSSによる段組ページ・レイアウトのモックアップがリストアップされている、CSS Depotというページを見つけました。カラムのサイズが可変かどうかとかヘッダーがあるかどうか、フッターがあるかどうかなどで5つのサンプルが公開されています。

CSSによる段組ページ・レイアウトはいろいろなところで触れられているのである程度は知識がある人が多いでしょうし、テクニック的にも特に目新しくは無いので、んー？　という感じがしたのですが、こういう風に並べてもらえるとわかりやすいですね。

5番目の2 columns, fixed size, centeredは結構面白いかも。文字サイズを変えるとカチャカチャレイアウトが変わるのでうざったいかもしれないですが。

このウェブサイトは、ながしまきょうがCC-BYの下、公開、更新しています。更新情報は、RSSやTwitterで購読できます。雑多な記録だけのRSSもあります。
