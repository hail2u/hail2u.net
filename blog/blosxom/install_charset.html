Install charset plugin

実はWriteBackの文字化けのみにしか使ってないという・・・。content_type.{flavour}とかあるし、head.{flavour}にmetaタグ書く場合はベタに記述して特に問題ないし。

blosxom本体にも文字コード変換を組み込んでも良さそうだけど、ただでさえ重いのが更に・・・。
