npmパッケージの名前、npmアカウントの削除

やりたいことがあり、CSVファイルを処理するcsv-parseとcsv-stringifyパッケージを使ったツールを書き捨てようとしていた。npm installしたものの、全然使えるようにならず、30分くらいハマった。インストールもちゃんとできているはずなのに、なぜか使えなかった。

$ npm install --save css-parse css-stringify

こうしていて違うパッケージをインストールしていたことが原因だった。2つのパッケージ名を両方typoするというミスでひどい。両方とも1文字違いのパッケージが存在するのが悪い。

少し前にnpmのアカウントを消した。とってつけたように重要かもしれない情報を書いてしまった。色々アカウントを消したので、そのことについては月が変わったあたりに書きたい。
