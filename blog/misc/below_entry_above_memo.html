エントリ以下メモ以上

エントリにするほどでもないけれど、メモよりはもうちょっとゴチャゴチャ何かしら書きたいときとかありますよね？　多くの場合、1つのエントリにいくつかまとめてどっかーんと載せるわけですが、それってどうなのよ？　とか思う。

わけはない。好きにしろよ。

  • ClipName
  • 多分もっとも優れたパスをコピーするツール。普通に送ってメニューを出すよりも、SendToにいろいろなコマンドライン・オプションを指定（Windows 2000以降だけだけど）しておいたショートカットを利用するのが良いかな。とかなんでかこんな時間に思うわけですよ。
  • W3C Markup Validation Service
  • 0.6.5がリリースされ、リニューアル。
  • Google Print (BETA) (from mnot's Web log）
  • 名称がおかしい気がする。まぁなんとなくはわかるけど。Amazon対抗か。
  • Latest Windows Crash
  • 右のカラムにあります。こういうくだらない物が置いてあると良い感じがする。BXXgPeXXleとか飽きた。
  • Raelさんからメール
  • 「#blosxomはどこよ？」とメールが来た。読まれてるらしい。フフフ。てか、「日本語だけど英語でも歓迎」とか心に無いことを言ってしまいました。#blosxomに外人あふれたらゴメンナサイ。っつっても5人くらいしかいないんですが。あ、入った時は「the zen of blogging!」と叫ばない（noticeコマンド使います）とbanされます。ええ嘘ですよ。
  • Semantic Coding
  • 今さら感があるのは気のせいだろうか。でも大御所（多分）が言うと悪い影響はでなそうな気はする。
  • blosxom 3.0 plugin with docs
  • 放置してあるblosxom 3.0用のプラグインにドキュメント埋め込みました。なんかやたらと海外からアクセスあるので。インチキ英語。日本語では書かない。ほらネイティブだから、ネイティブ。誰も騙されない嘘を書いて楽しいのでしょうか？

とかズラズラ書きたかっただけのために、あんな導入にしてみました。無駄なことを。
