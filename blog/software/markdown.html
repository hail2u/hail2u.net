Markdown

いろいろなWikiでの記法を始めとして、HTMLに変換することを前提とした記法はいろいろあります。その中で、僕が最もtxtとして可読性が高いんじゃないかな？　とおもっているのがMarkdownです。そのMarkdownが1.0としてリリースされました。

このblogのエントリ書きには適当にでっち上げた秀丸マクロを使っているんですが、独立したHTMLのファイルを書くときなんかはMarkdownを使ってます。また、ローカルで使っているYukiWikiMiniを改造したWikiもMarkdown利用だったり。アーカイブに同梱されているMarkdownの記法で書かれているReadme.textを見てもわかるように、素のテキストでも違和感がまるで無いので、とても書きやすく愛用。

Dingusでテストできるので、興味ある人は試してみましょう。かなりサクサク書けると思いますよ。

ちなみにMarkdownはそのまんまでblosxomのプラグインとして動作させることも可能です。Movable Typeでも使えたりするらしい。「HTMLを自由自在に書ける魔法の記法！」というわけではないですが、blogのエントリのフォーマッターとしては上々の出来だと思います。

このウェブサイトはながしまきょうがCC-BYの下で作成、公開、そして更新しています。更新情報はRSS (記事だけのRSS)またはXで購読できます。その他、わからないことはもう少し詳しい情報を参照してください。
