PDFCreatorでEPSをSVGに変換する

EPSをSVGに変換するソリューションは色々あるんだけど、あんまり考えずに出来て安定してるものはないかと探した末にPDFCreatorに落ち着いている。内部でGhostscriptを使っているので変換はそこそこ信頼できそう。GUIはクソでゴミをコーティングしたみたいな感じだけど、一回変換したら終わりなので我慢して使ってあげている。

  1. PDFCreatorを起動
  2. EPSファイルをドロップ
  3. 開いたダイアログでは特に何もせずSave
  4. 保存ダイアログのファイルの種類でSVG Files (*.svg)を選択して保存

たまにfillでの色の指定がうまくいってなくてパースエラーを起こし、表示できないSVGファイルが出来ることがある。

<g  stroke='none'#1ab7ea>

おかしくなっていると言っても大体はこの程度なので、エディタでちまちま直す。

<g  stroke='none' fill='#1ab7ea'>

経験上、pathが変なになったとかそういうのはあまりないので、お手軽コンバーターとしては十分。
