FirefoxのuserChrome.cssネタ #6

Dec 3, 2004·Software

少し前にUnknownPlace.のFirefox userChrome.css ネタで、タブ上のアイコンでfavicon以外は消すというuserChrome.cssネタが公開されていて、これには結構ビビッと（怪電波が）来たわけですが、このままだと読み込み中だと言うことを示すアイコンも消えてしまうので、そこをハックしてみました。

タブ上のアイコンでfavicon以外は消しても読み込み中を示すアイコンは消さない

.tabbrowser-tabs tab:not([busy]) > .tab-icon:not([src]) {
  width:0 !important;
  list-style-image:none !important;
}

読み込み中であることを示すのがtab[busy]なので、それを否定してゴニョゴニョしてみました。後半のnot([src])は元ネタを作ったUnknownPlace.のtypesterさんに教えて貰ったり。すっきり！元ネタの方ではmin-widthも指定されていますが、デフォルトのテーマでは指定する必要は無さそうです。

無闇に楽しい、userChrome.cssネタ。今回のネタは風呂に入っている時に思いつきました。えぇっと・・病気？
