2016年1月1日

vim-css3-syntax v0.16.0

CSSOM View Moduleで定義されているscroll-behaviorプロパティーを追加し忘れていたのを修正して、vim-css3-syntaxのv0.16.0をリリースした。このプロパティーはスムーズ・スクロールを制御するためのもので、自前でコンテンツを動かす場合にスムーズ・スクロールと競合しやすいことを抑制できる。実装は少なくないが、利用できる環境は少ない。

プラグイン名については色々あったが、2015/10/13版のスナップショットでは「もはやCSSレベル4は存在しない」と明言された。各モジュールレベルでは4やそれ以上になるかもしれないが、CSSからはレベルという概念はなくなるということらしい。CSS3という言葉は安定バージョン（＝スナップショットということのようだ）と策定中の非安定仕様との差分を意味することになる。

このままの名前で大丈夫そうで良かった。
