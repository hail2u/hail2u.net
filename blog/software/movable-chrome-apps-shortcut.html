Chrome 30のアプリのショートカットを動かせるようにする

Oct 19, 2013 · Software

Chrome 30で、ブックマーク・バーにアプリのショートカットが表示されるようになった。ブックマークではないので、動かせない。右クリック等から非表示にはできるけど、そうすると今度はアプリにアクセスする手段がなくなる。どうしたもんかと思ったけど、普通のタブなので改めてブックマークしてやれば良いだけだった。

アプリのショートカット、またはアドレス・バーから以下のURLを入力して、アプリのページを開く。

chrome://apps

アドレス・バーの☆をクリックし、ブックマークとして保存するだけ。アイコンも反映されるので、識別は難しくない。僕はあんまり使わないので「その他のブックマーク」フォルダに置いている。
