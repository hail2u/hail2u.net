vim-css3-syntax v0.7

on May 27, 2013 under Software

サボりすぎた。増えたのとか更新を反映させてvim-css3-syntaxをv0.7に更新した。同梱して欲しい要望があったのでafter/syntax/html.css等を同梱するようにも。問題起こる環境ありそう。なにかあったらIssuesでどうぞ。

CSS Conditional Rules Module Level 3のサポートはとりあえずなもの。@supportsとその中がだいたい上手くいくんじゃないかな程度であんまり考えてない。CSS Paged Media Module Level 3はAddとか言って空ファイル。ネスト絡みでちょっとすごく面倒そうなので誰か書いてくれないかなーと。他はまぁキーワードをバシバシ追加しただけで、テストを見る限りは大丈夫そう。

あ、死んでないし、誰か殺してもいないのでだいじょうぶです！更新なくなると犯罪犯したと思われる世知辛さ。ついでにFeedBurner使うのやめた話書こうかと思ったけど、トピック全然違うのでまた今度書く。削除後のリダイレクトは30日限定じゃなくて恒久になったので、安心して使うのやめられるよ！
