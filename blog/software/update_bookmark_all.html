Firefox Extension: Bookmark All 1.1

←

Bookmark Allを使ってくれた方が「フォルダ名を指定できると良いな」とかいう反応をすることが多いみたい（超他人事）なので、設定ダイアログの作り方のお勉強も兼ねてOption設定でフォルダ名のprefixを設定できるようにして、1.1に更新してみました。私は自分に都合の良い名前を付けてくれるように作ったわけなので、どうでも良かったりとかなんとか。

prefixのみの指定に制限されている理由は、自由に設定できるようにするとなると、結構メンドウなことになりそうな予感がしたとかそんな投げやりな理由だったりします。常にユニークなフォルダ名で作れるような仕組み（かぶりそうな場合は" (1)"とかpostfixを自動で付けるとか）を考えるのはメンドウだし、かといって同じ名前でずらっと並べられても困るだろうと。・・・ということにしておきます。あんまり意味無いかも。

なんか上手い方法ないかなぁ。とか書いたら誰か考えてくれるかな。

てか、私はあまりタブを大量に開かないので、滅多にBookmark Allを使わないわけで、コレで良いよとか思ったり思わなかったり。
