Shell Object Editor

Apr 29, 2003 · Software

Virtual Plasticより。Shell Object Editorという、My Computerやデスクトップにオブジェクトを作成するソフトウェア。ココによく使うフォルダを登録すれば、シェルのコンボボックスにでてくるので、コモンダイアログでアクセスしやすくなったりする。それ以外に利用価値は思いつかないけど。それなりに便利だと。それ以上にこういうこと思いつく柔軟な思考に萌え。
