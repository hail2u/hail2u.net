Desktop Sidebar

Desktop Sidebarは、デスクトップにサイドバーを作り、そこにいろいろなもの（システムの状態とかカレンダー、天気予報とか）を表示するアプリケーションです（そのまんま）。

MSN MessengerやWindows Media Playerとかぶち込めたりとなかなか怪しげでいい感じですね。でも、使おうとは思わないなぁ。なんか激しく不安定になりそうで。

そんなものをなぜ紹介したかって言うと、RSSに対応したからです。RSS Autodiscoveryにも。

それだけ。

てか、Longhornってこんな感じになるんでしたっけ？　こういうのって便利なんだろうか。いまいち・・・なぁ・・・。
