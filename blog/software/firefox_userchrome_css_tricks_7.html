FirefoxのuserChrome.cssネタ #7

ExplorerやInternet Explorerではメニュー・バーやロケーション・バー、ツールバーなどを比較的柔軟に配置できます（いわゆるCoolBarとかいうシロモノ）。Firefoxでもそれは同じ・・・どころか独自のツール・バーを作成できたりと、より多機能だったりします（独自のツール・バーを作成して使ってる人は少なそうだけど）。が、上下（メニュー・バーは下のみ）にborderが付いているだけと、CoolBarとは見た目が微妙に違います。ので、CoolBarっぽくしてやろうと。

ツール・バー全体を枠でくくる

#navigator-toolbox {
  border-top:1px solid threedshadow !important;
  border-right:1px solid threedhighlight !important;
  border-bottom:1px solid threedhighlight !important;
  border-left:1px solid threedshadow !important;
}

#toolbar-menubar,
#nav-bar,
#PersonalToolbar {
  margin:0 !important;
  padding:0 !important;
  border-top:1px solid threedhighlight !important;
  border-right:1px solid threedshadow !important;
  border-bottom:1px solid threedshadow !important;
  border-left:1px solid threedhighlight !important;
}

Explorerマンセー！ちょっと左下の角部分が不恰好ですが、これはGeckoのborder要素の仕様っぽい感じ。独自プロパティを駆使すれば本当にExplorerまんまなことも出来そうですが・・・（マンドクセ）・・・。

こんな感じになります。#nav-barへの指定は冗長だったりするけど気にしない。

そろそろネタ切れです。JavaScript ConcoleやDOM Inspectorのカスタマイズとかアレすぎなネタはあるのですが。そのうち書いちゃいそうだけど。

Firefoxまとめサイトからアクセスがずいぶん増えました。1.0リリース以前は20/dayくらいだったのが、1.0リリース以降50/dayくらいになり、最近は100/day近くにまで増えたり。そのこともあって、Creative Commonsで一部だけパブリック・ドメインにするとか機械的に出来ないかなとか考えてます。blog全体にCreative Commonsのあるライセンスを適用することはちと微妙とか最近になって考えてたりする（あんまり真面目には考えてないけど）ので、「この画像は自由に使ってイイヨ！」とか「このコードは好きにして・・・」とかやりたいとか思った。例えば、こういったuserChrome.cssネタを無断で転載できるようにCSSのコード部分だけをパブリック・ドメインにしたいとかそういうお話。
