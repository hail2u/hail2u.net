Background Color Disappearance on Edge 15

要素の背景色をカスタム・プロパティーを使って指定していて、かつその要素に擬似要素を追加し、かつその擬似要素で背景色を指定した場合、要素の背景色が初期値（通常はtransparent）に戻ってしまうバグのテスト・ページです。Edge 15でのみ再現します。

Lorem Ipsum

:root {
  --test: green;
}

.test {
  background-color: var(--test);
}

.test::before {
  background-color: #f0f;
  content: "Test";
}

この場合、.testの背景色はgreenになるべきだが、Edge 15ではtransparentになってしまう。Chrome 58やFirefox 53、Mobile Safari 10.3.2、Chrome 61、Firefox 55では想定通りgreenになる。
