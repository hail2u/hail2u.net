The Asterisk Hack

プロパティ名の頭にアスタリスクを付けるとInternet Explorerではアスタリスクを無視してしまう。

p.test1 {
  *color: #009900;
}

If this text is green, the browser has applied the rule.

Internet Explorer 7までの場合は文字が緑色になる。

付記

Underscore HackはInternet Explorer 7では通用しなくなっている。

p.test2 {
  _color: #009900;
}

If this text is green, the browser has applied the rule.

Internet Explorer 6までの場合は文字が緑色になる。
