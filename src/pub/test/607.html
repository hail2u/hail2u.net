Character Reference and Attribute Selector

文字参照を含む値をもつ属性に対して属性セレクターで要素を選択する場合、文字参照を戻した形で書き、必要ならばバックスラッシュを使ってエスケープする。

Test case for "

The title attribute value of this element is: &quot;foo&quot;


.test p[title = "foo"],
.test p[title = ""foo""],
.test p[title = "foo"] {
  color: red;
}

.test p[title = '"foo"'],
.test p[title = \"foo\"],
.test p[title = "\"foo\""] {
  color: green;
}
