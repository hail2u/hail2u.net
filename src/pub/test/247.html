FF Nuvo Web

February 24, 2010 — The latest FontFont release marks the beginning of a new era for typography – FSI FontShop International is introducing the first ever stand-alone FontFonts for the web. Finally web designers can use professional fonts for their projects without relying on system fonts or webfont services. Almost 40 of the most successful FontFont Families are available from today as Web FontFonts, including FF DIN Web, FF Meta Web Pro, FF Dax Web Pro, or FF Kievit Web Pro. Among the first Web FontFonts is Siegfried Rückel’s FF Nuvo, a contemporary type family with vertical contrast. You can now download FF Nuvo Web Medium for free for a limited time (Standard End User Licence Agreement for Web FontFonts applies).
