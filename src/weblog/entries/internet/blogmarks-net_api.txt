BlogMarks.net API
<p>サムネイル付きソーシャル・ブックマーク・サービスの<a href="http://blogmarks.net/" title="BlogMarks.net : Last public marks">BlogMarks.net</a>が<a href="http://dev.blogmarks.net/cgi-bin/blogmarks.cgi/wiki/DeveloperDocs" title="DeveloperDocs - Blogmarks - Trac">APIを公開しました</a>。流行りのXML over HTTPなAPIで、<a href="http://www.atomenabled.org/" title="AtomEnabled.org">Atom</a>で返答があるというモノ。</p>
<p>APIの公開自体は特に何も言うことは無いのですが、このサービスの特徴であるサムネイルの画像も取得できるところがものすごくポイント高いと思います。ぶっちゃけ任意のURLのサムネイルを拾ってくることが可能。つまり今までは<a href="http://www.hirax.net/pukiwiki/pukiwiki.php?cmd=read&amp;page=webnail2" title="webnail2 - hirax.net">webnail2</a>やその他プログラムでローカルでやっていたことが、かなり手軽に、かつ、ローカルのリソースを消費することなく実現できるというわけ。</p>
<p>実際には自分のアカウントにPOSTして、それをGETという感じになるっぽいので、URLを指定してサムネイルだけをガバッとゲットとはいかないみたいです。</p>
<p>なんにしても上手く使えばかなり面白そう。</p>
