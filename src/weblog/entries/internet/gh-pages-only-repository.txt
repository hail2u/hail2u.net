gh-pagesのみのリポジトリ

<p><a href="http://hail2u.github.io/blosxom-docs-ja/">blosxomサイトの日本語訳</a>を<a href="https://github.com/hail2u/blosxom-docs-ja">単独のリポジトリ</a>にした。URLはかわらないのであんまり気にすることはない。その時にやった<code>gh-pages</code>ブランチだけのリポジトリを作る手順のメモ。</p>

<p>まずGitHub上で<code>blosxom-docs-ja</code>リポジトリを作成する。</p>

<pre><samp>$ <kbd>cd ~/Projects/blosxom-docs-ja</kbd>
$ <kbd>git init</kbd>
$ <kbd>git branch gh-pages</kbd>
$ <kbd>git remote add origin git@github.com:hail2u/blosxom-docs-ja</kbd>
$ <kbd>git push --all</kbd>
</samp></pre>

<p>ここでGitHubのリポジトリの管理画面を開いて<code>gh-pages</code>をDefault Branchにする。</p>

<pre><samp>$ <kbd>git push origin :master</kbd>
$ <kbd>git checkout gh-pages</kbd>
$ <kbd>git branch -d master</kbd>
$ <kbd>git push -u origin gh-pages</kbd>
</samp></pre>

<p>これで<code>master</code>ブランチがない<code>gh-pages</code>ブランチだけのリポジトリになる。ググらずにやってできちゃったので不安なことこの上ない。</p>
