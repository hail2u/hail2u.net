CSSWring v5.1.0

<p>冗長なユニバーサル・セレクターを削除する機能を追加して、<a href="https://github.com/hail2u/node-csswring/releases/tag/v5.1.0">CSSWring v5.1.0</a>をリリースした。かなり前にやろうと考えていたもののすっかり忘れていた。v5.1.0には少し前に書いた単位の変換機能も含まれている。</p>

<p>冗長なユニバーサル・セレクターとは、省略可能な<code>*</code>のことだ。</p>

<pre><code class="language-css">*#foo,
*.bar,
*:link,
*[baz] {
  color: red;
}
</code></pre>

<p>このような<code>*</code>は、あってもなくてもセレクターの解釈へ影響を与えることはないため、削除できる。</p>

<pre><code class="language-css">.foo * .bar {
  color: red;
}
</code></pre>

<p>実際にはこのような<code>*</code>も削除可能だ。ただ実装がかなり面倒くさい。例えば以下のようなセレクターでは削除することはできない。</p>

<pre><code class="language-css">.foo &gt; * + .bar {
  color: red;
}
</code></pre>

<p>前後が<a href="https://www.w3.org/TR/css3-selectors/#simple-selectors">シンプル・セレクター</a>でなければならないというわけだ。シンプル・セレクターかどうかを判定する機能をうまく実装出来たら、削除できるようにしたい。</p>
