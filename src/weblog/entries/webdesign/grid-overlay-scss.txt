SCSSでグリッド・オーバーレイを簡単に

<p><a href="/blog/webdesign/grid-overlay-the-lazy-way.html">少し前に書いた</a><code>::before</code>擬似要素とCSSグラデーション、<code>pointer-events</code>プロパティを利用したグリッド・オーバーレイを<a href="https://github.com/hail2u/scss-partials/blob/master/_grid-overlay.scss">SCSSのミックスイン</a>としてみました。まぁグリッドじゃなくてカラムですけどね！</p>

<pre><code class="language-scss">@import "grid-overlay";

body {
  @include grid-overlay(60px, 20px);
}
</code></pre>

<p>という風にインポートしてカラムと溝(カラムとカラムの間)の幅を引数として書くだけです。この例では<a href="http://960.gs/">960.gs</a>と同じ間隔のグリッドを表示します。引数を省略した場合はカラムが<code>48px</code>で溝が<code>16px</code>のグリッドになります。<code>body</code>要素以外でも使えると思いますが、あんまり意味はないと思います。</p>

<p>リファレンスにもしっかり書いてありますが、ミックスインでも普通にネストしてセレクタを書くことが出来るので、</p>

<pre><code class="language-scss">@mixin foo() {
  &amp;::before {
    content: "Before Pseudo Element!";
  }
}
</code></pre>

<p>とか出来ることは憶えておくとちょっと便利です。</p>
