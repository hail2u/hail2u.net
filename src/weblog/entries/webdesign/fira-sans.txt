Fira SansのWebフォント

<p>Firefox OSに採用されているオープンソースのFira SansのWebフォントが<a href="http://mozilla.github.io/Fira/">MozillaのGitHub Pages経由で配信されていること</a>を知った。ちょっと前までGitHub Pagesはパフォーマンスに難があったけど、最近大幅に改善されたので、Fira Sansをウェブページで使いたい場合はこれを直接読み込めば良さそう。</p>

<p class="seealso">View Demo: <a href="/pub/test/536.html">Fira</a></p>

<p>CSSは<a href="https://github.com/mozilla/Fira/blob/gh-pages/fira.css">全部入りの</a>が用意されてるので、それを使うと簡単だけど、HTTPリクエストが増える。そのため以下のように必要なだけ直接<code>@font-face</code>ルールを自分で書く方が良い。</p>

<pre><code class="language-css">@font-face {
  font-family: &quot;Fira Sans&quot;;
  font-style: normal;
  font-weight: 400;
  src: local(&quot;Fira Sans&quot;),
    local(&quot;FiraSans-Regular&quot;),
    url(&quot;http://mozilla.github.io/Fira/woff/FiraSans-Regular.woff&quot;) format(&quot;woff&quot;);
}

@font-face {
  font-family: &quot;Fira Mono&quot;;
  font-style: normal;
  font-weight: 400;
  src: local(&quot;Fira Mono&quot;),
    local(&quot;FiraMono-Regular&quot;),
    url(&quot;http://mozilla.github.io/Fira/woff/FiraMono-Regular.woff&quot;) format(&quot;woff&quot;);
}
</code></pre>

<p>Google Fontsと違って固定URLでフォントが配信されているので、メンテナンス不要。Fira Monoもあるのでセットで使うとより良さそう。</p>

<hr>

<p>メイリオとの相性だとSource Sans Proの方が若干良いと思うけど、大体は良好な読みやすさ。コストなしで利用できるサンセリフのフォントとしてはトップクラスなので、去年Source Sans Pro使うページが増えたように、今年はFira Sans使うページが増えそうな気もする。</p>
