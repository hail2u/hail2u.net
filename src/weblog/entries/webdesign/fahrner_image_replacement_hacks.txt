Fahrner Image Replacement hacks
<p><a href="http://www.stopdesign.com/also/articles/replace_text/" title="Stopdesign | Using Background-Image to Replace Text">Fahrner Image Replacement</a>という、ロゴに画像を使用しつつ、CSSが切られていたりCSS非対応なブラウザには相応のテキストを表示するCSSテクニックがあるんです。<code>img</code>要素の<code>alt</code>属性使えばいいじゃんとかそういうのはまぁ置いておいて(一応、上記URLで触れられているので、そちら参照)。コンセプト的にはかなりツボなのですが、このテクニックはあんまり意味のないマークアップを施さなきゃならない上に、このままではロゴにリンクを張るのは難しい(<code>display:hidden;</code>なのでポイントできない)、さらには<a href="http://www.alistapart.com/articles/fir/" title="Facts and Opinion About Fahrner Image Replacement: A List Apart">A List Apartでアクセシブルっぽく見えて実際にはアクセシブルじゃないよとかつっこまれている</a>ので、ちょっとどうかなとか思っていたのですが、うまいこと追加マークアップ無しで実現できたので、このテクニックを応用したものをタイトル・ロゴに導入してみました。</p>
<p>まずはCSSのコード。</p>
<pre><code class="language-css">h1 {
  border-top:1px solid rgb(51, 51, 51);
  border-right:1px solid rgb(51, 51, 51);
  border-left:1px solid rgb(51, 51, 51);
  color:rgb(51, 51, 51);
  background-color:rgb(186, 192, 195);
  background-image:url(&quot;../images/titlelogo.png&quot;);
  background-position:4px 50%;
  background-repeat:no-repeat;
}

h1 a:link, h1 a:visited, h1 a:active, h1 a:hover {
  display:block;
  width:auto;
  height:42px;
  color:inherit;
  background-color:transparent;
  text-indent:-1000px;
  text-decoration:none;
}
</code></pre>
<p><code>h1</code>要素に、ロゴに使う画像を背景画像として指定しておきます。次に、<a href="http://www.w3.org/TR/CSS2/selector.html#descendant-selectors" title="Selectors">子孫セレクタ</a>を利用して、テキストを表示させないようにがんばってます。キモは<code>text-indent</code>で、これに大きめの負の値を入れてやることによって、テキストを見えないところにすっ飛ばしてやってます(See Also: <a href="http://phark.typepad.com/phark/2003/08/accessible_imag.html" title="Phark: Accessible Image Replacement">Accessible Image Replacement</a>)。これだけだと<code>h1</code>要素の大きさが不定になるので、<code>h1</code>要素の大きさを確保してやれば完成。<code>display</code>と<code>width</code>、<code>height</code>がそのための指定です。</p>
<p>Validで、適度にクロス・ブラウザ(Internet Explorer 6、Mozilla 1.6、Opera 7だけですが)で、適度にアクセシブルなので自己満足度は高いテクニックになりました。微妙にくだらないけど。</p>
<p>ちなみに、タイトル・ロゴの画像に使用したフォントは、<del>Zurich Light Extra Condensed BT</del><ins>Linotype Aroma Semibold</ins>です。かなり好き。</p>
