Google Web Fonts Previewを更新した

<p><a href="http://www.google.com/webfonts">Google Web Fonts</a>のHTMLが大幅に変わったため動かなくなっていた<a href="http://hail2u.github.io/google-webfonts-preview.html">Google Web Fonts Preview</a>を<a href="http://code.google.com/apis/webfonts/docs/developer_api.html">Google Web Fonts API</a>を使って書き直しました。だいたい動いてる感じ。便利ですよねー(棒読み)。</p>

<p>APIが返すJSONの読み込みは馬鹿のひとつ覚えでYQLを使いました。Webフォントの情報は、</p>

<pre><code class="language-js">{
  &quot;kind&quot;: &quot;webfonts#webfont&quot;,
  &quot;family&quot;: &quot;Anonymous Pro&quot;,
  &quot;variants&quot;: [
    &quot;regular&quot;,
    &quot;italic&quot;,
    &quot;bold&quot;,
    &quot;bolditalic&quot;
  ],
  &quot;subsets&quot;: [
    &quot;cyrillic&quot;,
    &quot;cyrillic-ext&quot;,
    &quot;greek&quot;,
    &quot;greek-ext&quot;,
    &quot;latin&quot;,
    &quot;latin-ext&quot;
  ]
}
</code></pre>

<p>こういったシンプルな構造で、Webフォントを参照するためのCSSのURLとかは自前で生成する必要があるようです。と言っても<code>family</code>の半角空白を<code>+</code>に置換するくらいですけど。<code>variants</code>や<code>subsets</code>は1つしかない場合でもちゃんとArrayですけど、YQL経由で読むとArrayじゃなくなったりする落とし穴があるので気をつけましょう(落ちました)。</p>
