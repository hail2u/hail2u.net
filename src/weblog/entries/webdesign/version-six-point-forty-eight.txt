<p>昔の「なんでもセンタリングで！」みたいなのを思い出すからなのか、どうもセンタリングが嫌いなようだ。久しぶりにセンタリングしまくる感じにしたらすぐに嫌になった。でかい画像にセンタリングで文字を重ねるみたいなのが流行っているのもありそう。</p>

<p>センタリングでかっこよくするには、少なくともバランス良く改行させる必要があるようだ。欧文なら非改行スペースを駆使して改行をコントロールすることができるけど、和文ではできない。かといっていちいち<code>span</code>要素でグループ化して改行しない固まりを指定していくのも無理がある。将来的には<code>text-wrap: avoid</code>である程度は制御できるようになる気がするので、それまでは<a href="http://adobe-webplatform.github.io/balance-text/">JavaScriptで文を真ん中あたりで分けるみたいなの</a>使う(作る)必要がありそう。</p>

<p>ほとんどすべて左寄せに戻したらホッとした。センタリングを想定して余白を確保していた画像の修正をちまちまするのが一番面倒だった気がする。</p>

<pre><code class="language-scss">@function pow($base, $exponent) {
  $raised: 1;

  @for $i from 1 through $exponent {
    $raised: $raised * $base;
  }

  @return $raised;
}

$ratio-golden: 1.618;

$width-content: (1 + 1 / pow($ratio-golden, 3));
$width-suppli: (1 / pow($ratio-golden, 2));
`
</code></pre>

<p>フィボナッチ数列使って作ろうかと思ったので<a href="https://github.com/hail2u/scss-functions/blob/master/number/_fibonacci.scss"><code>fibonacci()</code></a>を作ったけど、結局普通に黄金比で作った。</p>
