Source [Sans|Code] Pro

<figure>
  <img src="/images/blog/source-sans-code-pro.png" alt="Source Sans Pro / Source Code Pro">
</figure>

<p><a href="http://blogs.adobe.com/typblography/2012/08/source-sans-pro.html">Source Sans Pro</a>を本文に、<a href="https://blogs.adobe.com/typblography/2012/09/source-code-pro.html">Source Code Pro</a>をコードに使うようにした。Source Sans Proは400と700をそれぞれitalic付きで、Source Code Proは400と600を使ってる。共にGoogle Fontsを利用しているけど、WOFFのみをダウンロードするようにしておいた。</p>

<p>CalibriもNeue Helveticaも別に悪くはないのだけど、さすがに見飽きたので何かWebフォントにしようと色々試していた。Source Sans Pro以外には<a href="http://www.anatoletype.net/fonts/dejarip">DejaWeb</a>と<a href="http://www.tipografiagandhi.com/">Gandhi Sans</a>、<a href="http://sorkintype.com/fonts.html#mws">Merriweather Sans</a>が好感触だった。どれも欠点らしい欠点はないと思うのでどれでも良かったんだけど、ロゴにも使ったことだしとSource Sans Proに決めた。Source Code Proはなんとなく。</p>
