YourTotalSite
<p><a href="http://www.yourtotalsite.com/" title="YourTotalSite.com / helping you develop the web. better.">YourTotalSite</a>は「helping you develop the web. better.」てな感じで、ウェブサイトをより良くするための技術情報などをまとめたサイトです(via <a href="http://www.digital-web.com/" title="Digital Web Magazine">Digital Web Magazine</a>)。どちらかというとアクセシビリティーやユーザビリティにフォーカスしているらしく、Hack的なモノはない感じです。記事のスタイルはblog + linkblogですね。</p>
<p>blogの方は、記事ごとのページが、最初に要約があり、その先に詳細な本文が載るという書籍や大手ニュースサイトに近いつくりで、時間をかけて記事を作っている感じがして好感。ASPなオリジナルのシステムっぽい。が、広告が本文内に<code>float</code>されており、本文が半角30文字分くらいになってしまっているのはちと見づらい。サイドバーがスカスカなので、そっちに持っていっても良い気がする(CTR下がるけど)。ちなみに、RSSは要約のみ(要約がしっかりしているので悪くは無いです)。</p>
<p>linkblogの方は淡々とクリップされただけな感じ。blogのRSSとは別に配信されているのは個人的にはマイナスな感じ。</p>
<p>内容は様々。「いまさら・・・」みたいな感じを受けるものもあれば、「ホホゥ」と思えるものまで。スバラシイサイトになるかどうかは、結局、これからの更新にかかっているような。</p>
