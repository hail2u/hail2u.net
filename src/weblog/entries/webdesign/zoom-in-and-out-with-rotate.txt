回転しながらズームイン・ズームアウト

<p>オブジェクトを回転しながらズームさせるにはCSS3の<code>transition</code>と<code>transform</code>を組み合わせるだけで良い。ぐるぐるぐるぐるどっか～ん。Ch9とFx4b9、Op11、Sf5あたりなら動く。</p>

<p class="seealso">View Demo: <a href="/pub/test/142.html" class="rotate">Rotate and Zoom</a></p>

<p>CSSコードはややこしくはないが、<code>img</code>のリサイズをCSSでまかなったりマイナー寄りの書き方かもしれない。読みづらいのでベンダー拡張プリフィックスは省略した。実際に書く時はプロパティ名の頭に<code>-webkit-</code>/<code>-moz-</code>/<code>-o-</code>を付けてコピペすれば良い。</p>

<pre><code class="language-css">#zoom {
  width: 400px;
  height: 400px;
}

#zoom img {
  margin: 150px;
  border: none;
  display: block;
  width: 100px;
  height: 100px;
  transition-duration: 3s;
}

#zoom img:hover {
  margin: 0;
  width: 400px;
  height: 400px;
  transition-duration: 3s;
  transform: rotate(1080deg);
}
</code></pre>

<p>キモは<code>transform: rotate(1080deg);</code>で、1080度(三回転分)を指定することによってぐるぐる度を増す。<code>transition-duration</code>でアニメーションの長さを調節できる。あとは通常時は中央に小さく表示し、<code>:hover</code>で大きくなるように<code>margin</code>と<code>width</code>、<code>height</code>を適当に指定してやれば良い。</p>

<p>クリックすると回転しながらズームアウトし消すとかをCSS3だけでできないかなーと思ったが<a href="/pub/test/143-2.html">難しそう</a>。<code>:active</code>を使った場合、クリック後ズームアウトまではいけるが消えることは絶対にない。消すスタイルを定義したクラス名を<code>onclick</code>で割り当てるようにしてやれば<a href="/pub/test/143.html">普通にできる</a>。</p>

<h2>追記</h2>

<p>CSS3の<code>:target</code>擬似クラスを利用すると状態の変化を固定することができるので、<a href="/pub/test/143-3.html">CSS3だけでクリックするとズームアウトして消すことができた</a>。</p>

<p>完全に忘れていた<code>:target</code>擬似クラスについては、Delicious経由で<a href="http://lab.simurai.com/css/toggle/">CSS3 toggle-button without JavaScript</a>を見て思い出した。このデモでは重ね合わせた<code>a</code>要素のうち1つを<code>pointer-event</code>を使って無効にすることによってボタンをトグルさせている。</p>
