page-break-*の話

<p>印刷向けスタイルシートを書いていて、<code>page-break-inside</code>プロパティを<code>pre</code>要素や<code>table</code>要素で使ったんですが、どうやらInternet ExplorerとOpera以外は対応してなかったり中途半端みたいですね(2011年10月現在)。</p>

<p><code>page-break-before</code>と<code>page-break-after</code>プロパティでは<code>always</code>しか使えなかったり、<code>page-break-inside</code>プロパティは全然な感じです。</p>

<pre><code class="language-css">@media print {
  h1, h2, h3, h4, h5, h6 {
    page-break-after: avoid;
  }

  blockquote, pre, table {
    page-break-inside: avoid;
  }
}
</code></pre>

<p>見出しの直後での改ページや引用や整形済みテキスト、表内での改ページを制御したかったんですが……。</p>
