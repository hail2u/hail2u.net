ITmediaがフィード配信を開始
<p><a href="http://www.itmedia.co.jp/" title="IT総合情報ポータル「ITmedia」Home">ITmedia</a>が<a href="http://www.itmedia.co.jp/info/rss/" title="ITmedia Information">ほぼ全てのコーナーでフィードの配信を始めた</a>ようなので、勝手に作っていたフィードを削除しました。RSS 0.91とRSS 1.0、RSS 2.0が用意されているとかいう無駄に豪華な感じですね。一応、RSS 1.0のフィードへリダイレクトさせておいたつもり。</p>
<p>それぞれのフィードの<code>title</code>に「最新記事一覧」とか付いてるのが邪魔とか思った。<code>description</code>に「最新記事の一覧であることの説明」が書いてあるのは構わない、というか書いてあるほうが良いと思いますけど、<code>title</code>にもあるのは冗長な気が。</p>
<p>各<code>item</code>は残念ながら全文配信ではないようですね(とか書いてみる)。残念だ！</p>
<p>何気に99%はテーブルレスなHTMLコードに。「横幅1024ピクセル以上が前提かつ本文が目立たないようなCSSはいただけないのでは？」とか思ってても言わない。ましてや「横幅728ピクセルの広告のために横幅1024ピクセル前提とかふざけてるの？」とか言えない。</p>
<pre><code class="language-css">table.vbody {
  width: 100% !important;
}

table.vbody div#logoline {
  width: 100% !important;
}

table.vbody div#banner {
  display: none !important;
}

table.vbody div.lc {
  margin-right: -20px !important;
}

table.vbody div.rc {
  display: none !important;
}

table.vbody div.l600 {
  margin-right: 20px !important;
}
</code></pre>
<p>とかユーザー・スタイルシート(FirefoxではuserContent.cssというファイル)に書いておくと良いかも。</p>

<p><a href="http://www.astroarts.co.jp/" title="AstroArts - アストロアーツ">アストロアーツ</a>もフィードの配信を始めたようなので、こちらも削除してリダイレクト。</p>
