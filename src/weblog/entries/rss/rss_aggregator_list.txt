RSS Aggregatorリスト
<p>とりあえずチェックしたクライアント側で動作するRSS Aggregatorsだけ、まとめようかなと。MacやLinuxでWebを徘徊することがほとんど無いので、Windowsで使えるものばかりになっているところは勘弁してください。</p>
<p>アプリケーション名の前の数字は、機能面の5段階評価での採点です。かなり適当ですので参考までに。評価基準は以下のとおり。</p>
<ul>
<li>一般的な気がするインターフェイスを備える</li>
<li>RSSファイルの追加・削除がしやすい</li>
<li>環境(RSSファイル提供元/アプリケーション実行環境)に優しい</li>
<li>UTF-8のRSSファイルが読め、表示が出来る(バイナリ/ソースをほんの少し書き換えれば済むものも可)</li>
<li>個人的な主観によるその他機能の評価</li>
</ul>
<p>ということで、事実上4点満点。1点しか取れなかったものは削除してます。また安定しているかとかは、考慮してないです。まだまだ若いカテゴリのアプリケーションだと思うので、そこらへんは大目に見てもいいような気がするので。</p>
<h2>追記</h2>
<p>日本のものと海外のものを分けました。</p>

<dt>5 <a href="http://glucose.dip.jp/">glucose</a> Windows Free</dt>
<dd>なじみやすいメーラーのような3ペイン型のGUI。「センサー」でRSSファイルの提供されていないサイトも同じように読める。</dd>
<dt>4 <a href="http://www.infomaker.jp/">Headline-Reader</a> Windows &#165;1500</dt>

<dt>5 <a href="http://www.witha.jp/HepCat/">HepCat</a> Windows Free</dt>
<dd>3ペイン+αなGUI。後発だけあり良く練られている感じ。ツリー部分が多少わかりにくいかも・・・とか無理やりケチをつけてみる。良いです。</dd>
<dt>5 <a href="http://www.gluesoft.co.jp/NewsGlue/">NewsGlue</a> .Net &#165;3000 (thanks for liki)</dt>
<dd>なじみやすいメーラーのような3ペイン型のGUI。リマインダや取得した記事をメールで転送など多機能。基本的には無料(らしい)。</dd>
<dt>3 <a href="http://fls.moo.jp/moz/rssreader.html">RSS Reader Panel</a> Mozilla Free</dt>
<dd>XULアプリケーション。ブラウザでRSSを読むことが出来る程度のもの。軽さやブラウザとの連携など見るべきところは多いが、やや機能に欠けるかも。</dd>
<dt>5 <a href="http://www.geocities.jp/nao_rssgate/">RSSGate</a> XSL(?) Free</dt>
<dd>ローカルサーバーとして動作する。閲覧はブラウザで、整形にはXSLを利用している模様。通常のウェブページもスクリプトを作成すればRSSにコンバートして閲覧できるなど、高機能。</dd>
<dt>5 <a href="http://testesca.at.infoseek.co.jp/hoka/sonota.htm#Rreader">RThReader</a> Windows Free</dt>
<dd>4ペイン型のGUI。挙動は俊敏。多少画面がごちゃごちゃしすぎている嫌いはあるが、整理できるので問題無し。</dd>
