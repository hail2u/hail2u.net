Bulkfeeds Similarity Search
<p><a href="http://bulkfeeds.net/" title="Bulkfeeds: Home - RSS Directory &amp; Search">Bulkfeeds</a>に新たな機能、<a href="http://blog.bulknews.net/mt/archives/000763.html" title="blog.bulknews.net: Bulkfeeds: Similarity Search リリース">Similarity Searchが加わりました</a>。かなりいろいろなところで取りあげられていますが、面白いように似た感じのエントリが拾えて楽しすぎるので、うちでもエントリを書いてみる。</p>
<p>Permanent linkをクエリに送ってやると、そのエントリから単語抽出してBulkfeedsのデータベースから探してくれるというモノのようです。あまり特徴的なことを書いていない適当なエントリの場合は<a href="/blog/webdesign/widgetopia.html" title="hail2u.net - Weblog - Widgetopia">思ってもみない結果を得ることがあります</a>が、特徴的な単語が入っている場合は<a href="/blog/media/incubus_a_crow_left_of_the_murder.html" title="hail2u.net - Weblog - Incubus / A Crow Left Of The Murder">かなり良い感じで結果を貰えます</a>ね。スゴイよ！</p>
<p><a href="http://www.blosxom.com/" title="blosxom :: the zen of blogging ::">blosxom</a>にこれを挿入したい場合、story.{flavour}に書くことになるわけですが、普通に書くと一気にBulkfeedsにアクセスしにいくことになるので、簡単なpluginでも書いて、エントリごとの閲覧の時のみ出てくるようにした方が良いかと。僕はすげー適当に書きました。</p>
<p>てか、どうやら僕はBulkfeedsのMenuExt作成係らしいので、作っておきました。手動でレジストリに追加してください。blogのテンプレで使った方が明らかに面白い(blogの管理人と訪問者共に)ので、BookmarkletやMenuExtは使う機会少なそうですが。</p>
<pre><code class="language-html">&lt;html&gt;
&lt;script type=&quot;text/javascript&quot; defer&gt;
&lt;!--
  var pWin = external.menuArguments;
  var uri = encodeURIComponent(pWin.location.href);
  uri = &quot;http://bulkfeeds.net/app/similar?url=&quot; + uri;
  pWin.window.open(uri, &quot;similaritySearch&quot;, &quot;&quot;);
//--&gt;
&lt;/script&gt;
&lt;/html&gt;
</code></pre>
