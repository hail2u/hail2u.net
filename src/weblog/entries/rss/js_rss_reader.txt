JScript RSS Reader
<p>なんとなく思いついたので<a href="/pub/js_rss_reader.html" title="JScript RSS Reader">JScript RSS Reader</a>なるものを作ってみました。セキュリティの都合上、インターネット上では動かないと思われる(動かそうと思えば動かせるのですが)ので、ローカルに<a href="/pub/js_rss_reader.html" title="JScript RSS Reader">このHTMLファイル</a>と<a href="/pub/js_rss_reader.xsl" title="JScript RSS Reader用のXSL Stylesheet">専用のXSL Stylesheet</a>も一緒に保存して、動かしてください。<a href="http://www.microsoft.com/japan/msdn/xml/downloads/msxml3sp2.asp" title="Microsoft XML Parser 3.0 SP2">Microsoft XML Parser 3.0 SP2</a>か<a href="http://www.microsoft.com/japan/msdn/xml/downloads/msxml4.asp" title="Microsoft XML Core Services (MSXML) 4.0 SP1 日本語版">Microsoft XML Core Services 4.0 SP1</a>が必要だと思います。</p>
<p>使い方は特に書くこともないような。</p>
<p>あーでもこれ面白いな。<a href="http://www.w3.org/TR/xslt" title="XSL Transformations (XSLT)">XSLT</a>とかごく稀にしか触らないので、忘れ去りかけてたのはヒミツです。つか、これを応用すれば<a href="http://www.microsoft.com/japan/msdn/scripting/default.asp" title="Windows Script">WSH</a>でRSS Headline Grabber程度のもの物なら作れますね(多分)。作らないですけど。</p>
<p>Active Desktopとか存在そのものが無かったことにされている機能と組み合わせても面白いかも。セレクトボックスにRSSファイルをずらーっと並べて、気が向いたら選択して表示させるとか。まさにプッシュ型メディア。・・・なんか違う。HTAとかこれまた無かったことにされている(以下略)。</p>
