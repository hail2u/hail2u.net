RssMemo公開
<p>RSSで保存するメモCGIとRSSで保存するメモCGIを使用開始してみるで言及したRssMemoを公開しました。いろいろなバグをつぶしたり、コードの整理をしたりして、初回パスワード設定がわかりにくい以外の点は、それなりに使えるスクリプトになったと思います。</p>
<p>詳細は<del>至れり尽くせリな</del><ins>あんまり詳細になってない</ins>同梱のreadme.txtを参照してください。動作環境はPerl 5.6で、必要なモジュールは、</p>
<ul>
<li><a href="http://search.cpan.org/search?module=CGI" title="search.cpan.org: CGI - Simple Common Gateway Interface Class">CGI</a></li>
<li><a href="http://search.cpan.org/search?module=Digest::MD5" title="search.cpan.org: Digest::MD5 - Perl interface to the MD5 Algorithm">Digest::MD5</a></li>
<li><a href="http://search.cpan.org/search?module=Jcode" title="search.cpan.org: Jcode - Japanese Charset Handler">Jcode</a></li>
<li><a href="http://search.cpan.org/search?module=XML::RSS" title="search.cpan.org: XML::RSS - creates and updates RSS files">XML::RSS</a></li>
</ul>
<p>です。多分、モジュールさえあれば動くと思う・・・な。</p>
<p>ToDoじゃないけど、対応したら良さげなことは、</p>
<ul>
<li>UTF-8以外での出力に対応</li>
<li>出力HTMLもカスタマイズ</li>
<li>RSS 0.92以外での出力</li>
<li>JavaScriptの出力の選択</li>
</ul>
<p>など。ログの保存はやりません。これはほぼ決定しました。RSSリーダーで保存していけば良いと思うので。あると良いかもとは少しだけ思うので、必要に迫られたら実装するかもしれないですけど。</p>
