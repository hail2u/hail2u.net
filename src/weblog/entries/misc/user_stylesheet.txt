ユーザー スタイルシート
<p>ユーザー スタイルシートに</p>
<pre><code class="language-css">body, table, tr, th, td, form, fieldset, legend, label, input, select, optgroup, option, textarea, button {
  font-family:Verdana, sans-serif;
}
</code></pre>
<p>とか書いて使用すると、Verdanaがデフォルトに設定できることにいまさら気づいた。OperaやMozillaのように、</p>
<pre><code class="language-css">img[width=468][height=60] {
  display:none;
}
</code></pre>
<p>というような属性セレクタに対応しているのなら、これで486*60のサイズであることが多い横長バナーを非表示に出来たり。属性セレクタには「～で始まる」とかもあるので、URIを指定して弾けそう。これ良いな。</p>
