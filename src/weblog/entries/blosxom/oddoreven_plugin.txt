oddoreven plugin
<p>久々。<a href="/archives/blugins.html#oddoreven" title="hail2u.net - Archives - blosxom plugins">oddoreven</a>というプラグインを作りました。これはそのエントリがページ内で奇数番目か偶数番目を知ることが出来るプラグインです。つまり、エントリのCSSルールを交互に変えるとかいうことが出来たりします。それ以外になんか使い道があるかどうかは知らないです。</p>
<p>具体的には、story.flavourを、</p>
<pre><code class="language-html">&lt;div class=&quot;section $oddoreven::str&quot;&gt;
  &lt;h3&gt;$title&lt;/h3&gt;
  $body
&lt;/div&gt;
</code></pre>
<p>というように、<code>class</code>属性に<code>$oddoreven::str</code>を指定してやって、CSSファイルに、</p>
<pre><code class="language-css">div.odd {
  color: #333333;
  background-color: #ddddff;
}

div.even {
  color: #333333;
  background-color: #ffdddd;
}
</code></pre>
<p>と、ルールを追加してやると奇数番目のエントリの背景は薄いブルー、偶数番目のエントリは薄いピンクになるっていうわけです。ちなみに、<a href="http://www.w3.org/TR/html401/struct/global.html#h-7.5.2" title="The global structure of an HTML document"><code>class</code>属性は半角空白で区切って複数の値をとることが出来ます</a>。</p>
<p>こんなのあったら地味に便利そうかなーとか思ったので30秒くらいで書いてみました。</p>
<p>バグを見つけたらBBSや<a href="http://www.freeml.com/info/blosxom@freeml.com" title="FreeML ML情報ページ [all about blosxom]">ML</a>で報告してくれれば対処すると思います。</p>
