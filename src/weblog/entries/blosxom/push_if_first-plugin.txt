push_if_first plugin
<p>二ヶ月ほど前にでっち上げた、最初のエントリにだけデータを追加できるようにする<a href="/archives/blugins.html#push_if_first" title="hail2u.net - Archives - blosxom plugins">push_if_first</a>というプラグインを<a href="/archives/blugins.html" title="hail2u.net - Archives - blosxom plugins">blosxom plugins</a>で公開しておきました。「最初のエントリだけに<a href="https://www.google.com/adsense/" title="Google AdSense">Google AdSense</a>が表示されていますが、どうやってるんですか？」というメールでの質問への返事がてら。</p>
<p>例えば、</p>
<ul>
<li>Google AdSenseを最初のエントリの後にだけ表示する</li>
<li>Amazonアソシエイトの商品紹介バナーを最初のエントリの後にだけ表示する</li>
<li>A8.net(以下略</li>
</ul>
<p>などということを実現するために利用できます。</p>
<p>こういったアサマシ用途以外にももちろん使い道はあります。<a href="http://www.blosxom.com/plugins/interpolate/interpolate_fancy.htm" title="blosxom :: the zen of blogging :: plugins/interpolate/interpolate_fancy.htm">interpolate_fancy</a>がインストールされているならば、<code>$push_if_first::is_first</code>という最初のエントリの出力が終わっていない場合のみ1を返す変数を利用すれば、「最初のエントリは全文表示だが、残りはpermalinkへのリンクが張られたタイトルとのみ」とかいうことも可能です(多分)。</p>
