FirefoxのuserChrome.cssネタ #6
<p>少し前に<a href="http://www.unknownplace.org/" title="UnknownPlace.">UnknownPlace.</a>の<a href="http://www.unknownplace.org/blosxom/software/firefox_userchrome_css.html" title="UnknownPlace. - Firefox userChrome.css ネタ">Firefox userChrome.css ネタ</a>で、タブ上のアイコンでfavicon以外は消すというuserChrome.cssネタが公開されていて、これには結構ビビッと(怪電波が)来たわけですが、このままだと読み込み中だと言うことを示すアイコンも消えてしまうので、そこをハックしてみました。</p>
<h2>タブ上のアイコンでfavicon以外は消しても読み込み中を示すアイコンは消さない</h2>
<pre><code class="language-css">.tabbrowser-tabs tab:not([busy]) &gt; .tab-icon:not([src]) {
  width:0 !important;
  list-style-image:none !important;
}
</code></pre>
<p>読み込み中であることを示すのが<code>tab[busy]</code>なので、それを否定してゴニョゴニョしてみました。後半の<code>not([src])</code>は元ネタを作ったUnknownPlace.のtypesterさんに教えて貰ったり。すっきり！元ネタの方では<code>min-width</code>も指定されていますが、デフォルトのテーマでは指定する必要は無さそうです。</p>
