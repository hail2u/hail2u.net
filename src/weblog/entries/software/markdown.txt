Markdown
<p>いろいろなWikiでの記法を始めとして、HTMLに変換することを前提とした記法はいろいろあります。その中で、僕が最もtxtとして可読性が高いんじゃないかな？とおもっているのが<a href="http://daringfireball.net/projects/markdown/" title="Daring Fireball: Markdown">Markdown</a>です。そのMarkdownが1.0としてリリースされました。</p>
<p>このblogのエントリ書きには適当にでっち上げた秀丸マクロを使っているんですが、独立したHTMLのファイルを書くときなんかはMarkdownを使ってます。また、ローカルで使っている<a href="http://www.hyuki.com/yukiwiki/mini/" title="YukiWikiMini Home">YukiWikiMini</a>を改造したWikiもMarkdown利用だったり。アーカイブに同梱されているMarkdownの記法で書かれているReadme.textを見てもわかるように、素のテキストでも違和感がまるで無いので、とても書きやすく愛用。</p>
<p><a href="http://daringfireball.net/projects/markdown/dingus" title="Daring Fireball: Markdown Web Dingus">Dingus</a>でテストできるので、興味ある人は試してみましょう。かなりサクサク書けると思いますよ。</p>
<p>ちなみにMarkdownはそのまんまで<a href="http://www.blosxom.com/" title="blosxom :: the zen of blogging ::">blosxom</a>のプラグインとして動作させることも可能です。<a href="http://www.movabletype.org/" title="Movable Type Publishing Platform">Movable Type</a>でも使えたりするらしい。「HTMLを自由自在に書ける魔法の記法！」というわけではないですが、blogのエントリのフォーマッターとしては上々の出来だと思います。</p>
