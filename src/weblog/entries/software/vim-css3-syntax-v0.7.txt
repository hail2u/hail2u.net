vim-css3-syntax v0.7

<p>サボりすぎた。増えたのとか更新を反映させて<a href="https://github.com/hail2u/vim-css3-syntax/tree/v0.7">vim-css3-syntaxをv0.7に更新した</a>。同梱して欲しい要望があったので<code>after/syntax/html.css</code>等を同梱するようにも。問題起こる環境ありそう。なにかあったらIssuesでどうぞ。</p>

<p><a href="http://www.w3.org/TR/css3-conditional/">CSS Conditional Rules Module Level 3</a>のサポートはとりあえずなもの。<code>@supports</code>とその中がだいたい上手くいくんじゃないかな程度であんまり考えてない。<a href="http://www.w3.org/TR/css3-page/">CSS Paged Media Module Level 3</a>はAddとか言って空ファイル。ネスト絡みでちょっとすごく面倒そうなので誰か書いてくれないかなーと。他はまぁキーワードをバシバシ追加しただけで、テストを見る限りは大丈夫そう。</p>
