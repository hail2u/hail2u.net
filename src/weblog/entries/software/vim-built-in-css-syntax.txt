Vim 7.4ビルトインのCSS構文ファイル

<p><a href="http://www.vim.org/">Vim</a> 7.4の標準ランタイムに含まれているCSS構文ファイルにはかなり大幅な変更が加わっている。<a href="http://www.w3.org/TR/css3-animations/">CSS Animations</a>等の新しいCSS仕様に対応していたり、非推奨になった要素を<code>Error</code>で表示するようになったり、ベンダー拡張プリフィックス部分を別に強調したりなどなど。</p>

<p>開発は<a href="https://github.com/JulesWang/css.vim">GitHubのリポジトリ</a>でやっているようなので、手元で修正してたものをプル・リクエストさせてもらった。こっちの方では既に大幅な変更が加わっている。ドラスティックな変更点だとCSSの値の強調色が変わってたりとか。他はCSS3モジュールへのさらなる対応などが主なもの。同梱バージョンではバグの修正はあまりされていないので、<code>text-indent</code>や<code>inline-table</code>でおかしくなるのが気になってた人は入れ替えると良い。</p>

<hr>

<p><a href="https://github.com/hail2u/vim-css-syntax">vim-css-syntax</a>での修正内容は取り込んでもらえ、こちらは用済みっぽいので中味を空にした。使っている人はアンインストールして、上記リポジトリで開発中のを代わりに入れると良い。<a href="https://github.com/hail2u/vim-css3-syntax">vim-css3-syntax</a>の方もぼちぼち開発中のに対応した形でスリムにしつつ、折を見てポートしていこうと思っている。</p>
