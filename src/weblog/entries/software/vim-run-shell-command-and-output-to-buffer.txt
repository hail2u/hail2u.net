Vimでちょっと実行してバッファーに吐く

<p><a href="http://www.vim.org/">Vim</a>でちょっとシェルのコマンドを実行してその結果をバッファーに持っておきたい時には<code>"=</code>レジスターと<code>put</code>を使ってやれば良いことがわかった。Windowsだとちょっと便利な気がする。他にもっと良い方法がありそうだけど、なんとなく面白いのでこれ使うことにした。</p>

<pre><code class="language-vim">:new | put =system(\"ls\")
</code></pre>

<p>で<code>ls</code>の結果が上に分割されて開く。引用符はエスケープ必須。<code>nofile</code>な感じにしたりとかしておいた方が色々と楽そうなので、ラップする関数とコマンド作った方が良さそう。</p>
