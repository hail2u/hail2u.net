Greenfish Icon Editor

<p><a href="http://icofx.ro/">IcoFX</a>が更新再開したらアレな感じになってたので、その代わりになるアイコン・エディターをずっと探してた。オンラインのものも含めていくつか良さそうなものがあったけど、IcoFXのようないわゆるアイコン・エディターが良かったので、触った感じが一番近かった<a href="http://greenfishsoftware.blogspot.jp/2012/07/greenfish-icon-editor-pro.html">Greenfish Icon Editor</a>に落ち着いた。</p>

<p>通常の簡単な編集作業に加え、レイヤー対応だったり、フィルターが充実していたり、<code>.icl</code> (<code>.dll</code>)や<code>*.icns</code>の編集も可能など多機能で、アイコン・エディターとして必要なものはすべて揃っているといった感じ。その一方でメニューやGUIウィジェットが整理されてて、単に他の画像エディターで作成した画像をアイコン化するGUIツールとしても使いやすい。</p>

<p>地味に便利だったのが、Photoshopライクに数グリッドごとにセル化してくれてルーラー不要な感じにしてくれるのがカスタマイズできる点。中心合わせをしやすいように、縦横にセンターラインを表示するだけのもある。</p>

<p>これ！といった特徴的な機能はないけど、大体において良くまとまっていて使いやすい。</p>
