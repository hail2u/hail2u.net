MSJVMを削除
<p><a href="http://www.microsoft.com/" title="Microsoft Corporation">Microsoft</a>の<a href="http://www.microsoft.com/mscorp/java/" title="Transitioning from the Microsoft Java Virtual Machine">Java Virtual Machine</a>を削除するツール、<a href="http://www.microsoft.com/downloads/details.aspx?familyid=f2002119-b4d5-4013-83bc-4a8ad95e959f&amp;displaylang=en" title="Download details: MSJVM Removal Tool 1.0">MSJVM Removal Tool</a>が公開されました。Microsoftから公式に出てるのがビックリするところ。</p>
<blockquote cite="http://www.microsoft.com/downloads/details.aspx?familyid=f2002119-b4d5-4013-83bc-4a8ad95e959f&amp;displaylang=en">
<p>The Microsoft JVM Removal Tool can be used to remove the MS Java Virtual Machine (MSJVM). Use of this tool is the only supported method for removing the MSJVM from a Microsoft operating system.</p>
</blockquote>
