git logでの%C(auto)

<p><code>git log</code>で<code>--oneline</code>だとコミットした人の名前やコミットされた日付が出ないので、昔頑張って<a href="/blog/software/easy-to-read-git-log.html"><code>--pretty</code>でキレイにする奴</a>を書いた。ただこれだとタグやブランチ名などが全部同じ色になっちゃってて、微妙に不便だった(<code>--oneline</code>では自動色分けされる)。我慢して使っていたんだけど、直せるのかなーと<a href="http://git-scm.com/docs/git-log">git-logのヘルプ</a>をちゃんと読んだら<code>%C(auto)</code>を使えば自動色分けされることがわかった。</p>

<pre><samp>$ <kbd>git log --graph --pretty=&#39;format:%C(yellow)%h&lt;mark&gt;%C(auto)%d%Creset&lt;/mark&gt; %s %C(black bold)%an, %ar%Creset&#39;</kbd>
</samp></pre>

<p><code>%C(auto)</code>が効くかどうかはプレースホルダーによる。ref nameである<code>%d</code>には効くけど、例えば<code>%an</code>や<code>%ar</code>などには効かない。</p>
