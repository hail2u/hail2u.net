IEDevToolbar
<p>Microsoftから<a href="http://www.microsoft.com/downloads/details.aspx?FamilyID=e59c3964-672d-4511-bb3e-2d5e1db91038&amp;displaylang=en" title="Download details: IEDevToolbar">IEDevToolbar</a>とかいう「Web開発に欠かせない！」とか「Web 2.0時代のマストアイテム」とか言ってもらいたそうなInternet Explorer用ツールバーのBetaが出ました。「<a href="https://addons.mozilla.org/extensions/moreinfo.php?application=firefox&amp;id=60" title="Mozilla Update :: Extensions -- More Info:Web Developer - All Releases">Web Developer</a>のパク・・・」とか言うとこの世から抹殺されそうなので言いません。</p>
<figure><img src="/images/blog/iedevtoolbar.png" alt="Internet Explorer Developer Toolbarのスクリーンショット"></figure>
<p>味も素っ気も無い見た目ですが、使い勝手は悪くないです。DOM Inspectorなどがあるので、トータルでは<a href="http://www.mozilla.org/products/firefox/" title="Firefox - Rediscover the web">Firefox</a>の圧勝でしょうけど、一通り機能は揃っているのでこれはこれで良いのかも。とは言うもののこれだけで他には何もいらないというわけではないので、意地でもInternet Explorerでという人は、</p>
<ul>
<li><a href="http://www.cheztabor.com/IEDocMon/index.htm" title="IEDocMon">IEDocMon</a>: DOM Inspectorみたいな奴</li>
<li><a href="http://www.blunck.info./iehttpheaders.html" title="blunck.info">ieHTTPHeaders</a>: Live HTTP Headersみたいな奴</li>
</ul>
<p>なども合わせて入れておくと良いかも。バッティングしそうで恐ろしいですが。</p>
<p>バグ報告とかは<a href="http://channel9.msdn.com/wiki/" title="Channel9 Wiki: HomePage">Channel9 Wiki</a>の<a href="http://channel9.msdn.com/wiki/default.aspx/Channel9.InternetExplorerDevToolbar" title="Channel9 Wiki: InternetExplorerDevToolbar">Internet Explorer Developers Toolbar Feedback</a>というページでやるみたい。</p>
