vim-css3-syntax v0.16.0

<p><a href="http://www.w3.org/TR/cssom-view/">CSSOM View Module</a>で定義されている<code>scroll-behavior</code>プロパティーを追加し忘れていたのを修正して、<a href="https://github.com/hail2u/vim-css3-syntax/releases/tag/v0.16.0">vim-css3-syntaxのv0.16.0</a>をリリースした。このプロパティーはスムーズ・スクロールを制御するためのもので、自前でコンテンツを動かす場合にスムーズ・スクロールと競合しやすいことを抑制できる。実装は少なくないが、利用できる環境は少ない。</p>

<aside class="pull-quote">
<p>There is no CSS Level 4.</p>
</aside>

<p>プラグイン名については色々あったが、<a href="http://www.w3.org/TR/2015/NOTE-css-2015-20151013/">2015/10/13版のスナップショット</a>では<q>もはやCSSレベル4は存在しない</q>と明言された。各モジュールレベルでは4やそれ以上になるかもしれないが、CSSからはレベルという概念はなくなるということらしい。CSS3という言葉は安定バージョン（＝スナップショットということのようだ）と策定中の非安定仕様との差分を意味することになる。</p>

<p>このままの名前で大丈夫そうで良かった。</p>
