vim-css3-syntax v0.11.0

<p><a href="https://github.com/hail2u/vim-css3-syntax">vim-css3-syntax</a>をv0.11.0にした。新たに追加したのは、文章の縦方向のリズムを簡単に制御できるようにするLine Gridモジュール、Web ComponentsへとつながることになるScopingモジュール、後にウニョウニョ動かすのでハードウェア・アクセラレーション有効にしてとかリフロー起こるかもみたいなのをブラウザーに教えるWill Changeモジュールの3種。</p>

<p>新規追加したモジュールの仕様は以下の通り。</p>

<ul>
<li><a href="http://www.w3.org/TR/css-line-grid-1/">CSS Line Grid Module Level 1</a></li>
<li><a href="http://www.w3.org/TR/css-scoping-1/">CSS Scoping Module Level 1</a></li>
<li><a href="http://www.w3.org/TR/css-will-change-1/">CSS Will Change Module Level 1</a></li>
</ul>

<p>Line Gridモジュールはサンプル見るだけでなんとなくわかり、かつ使いたい感じが高まる。タイポグラフィーに興味ある人は読んでみるとワクワクすると思う。</p>

<hr>

<p>最近はCSS仕様名の命名規則が変化してて、モジュールそれぞれでレベルが順序立てて名付けられるようになった。それらをまとめて「CSS3」になるんだけど、肝心のスナップショットが出ないので、CSS3とはなんだったのかみたいな。プラグイン名間違えた感じする……。</p>
