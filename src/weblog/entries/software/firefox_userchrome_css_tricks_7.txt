FirefoxのuserChrome.cssネタ #7
<p>ExplorerやInternet Explorerではメニュー・バーやロケーション・バー、ツールバーなどを比較的柔軟に配置できます(いわゆるCoolBarとかいうシロモノ)。<a href="http://www.mozilla.org/products/firefox/" title="Firefox - Rediscover the web">Firefox</a>でもそれは同じ・・・どころか独自のツール・バーを作成できたりと、より多機能だったりします(独自のツール・バーを作成して使ってる人は少なそうだけど)。が、上下(メニュー・バーは下のみ)に<code>border</code>が付いているだけと、CoolBarとは見た目が微妙に違います。ので、CoolBarっぽくしてやろうと。</p>
<h2>ツール・バー全体を枠でくくる</h2>
<pre><code class="language-css">#navigator-toolbox {
  border-top:1px solid threedshadow !important;
  border-right:1px solid threedhighlight !important;
  border-bottom:1px solid threedhighlight !important;
  border-left:1px solid threedshadow !important;
}

#toolbar-menubar,
#nav-bar,
#PersonalToolbar {
  margin:0 !important;
  padding:0 !important;
  border-top:1px solid threedhighlight !important;
  border-right:1px solid threedshadow !important;
  border-bottom:1px solid threedshadow !important;
  border-left:1px solid threedhighlight !important;
}
</code></pre>
<p>Explorerマンセー！ちょっと左下の角部分が不恰好ですが、これはGeckoの<code>border</code>要素の仕様っぽい感じ。独自プロパティを駆使すれば本当にExplorerまんまなことも出来そうですが・・・(マンドクセ)・・・。</p>
<figure><img src="/images/blog/toolbox_with_border.png" alt="ツール・バー全体を枠で括ってExplorerライクに"></figure>
<p>こんな感じになります。<code>#nav-bar</code>への指定は冗長だったりするけど気にしない。</p>
