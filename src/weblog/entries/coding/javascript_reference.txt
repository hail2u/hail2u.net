JavaScriptのリファレンス
<p>Microsoftの提供している<a href="http://www.microsoft.com/japan/msdn/scripting/default.asp" title="Windows Script">Windows Script</a>、その5.6のドキュメントではJScriptとなっていますが、JScriptはJavaScriptと文法は互換のなので、そのままリファレンスとして流用できます。HTMLヘルプなのでエディタから単語拾ってジャンプとかも結構魅力。JavaScriptギークにはマスト！とか。</p>
<p>また、今は亡きNetscape DevEdgeで配布していたCore JavaScript Reference 1.5は、<a href="http://web.archive.org/web/20040211210156/devedge.netscape.com/library/manuals/2000/javascript/1.5/reference/" title="Core JavaScript Reference 1.5: Core JavaScript Reference">Wayback MachineにZIPアーカイブ共々残っている模様</a>です。こちらはHTMLなので、どんな環境でも扱いやすく、英語ではありますがこちらも抑えておきたいところ。</p>
<p>以上の2つがあればJavaScriptのリファレンスとしては十分なんじゃないかなと。少なくとも基本は抑えられるはず。</p>
<p>今ではJavaScriptで何かやろうとすると避けては通れない<a href="http://www.w3.org/DOM/" title="W3C Document Object Model">DOM</a>のリファレンスは、これも今は亡きNetscape DevEdgeで配布していた<a href="http://www.mozilla.org/docs/dom/domref/" title="Gecko DOM Reference">Gecko DOM Reference</a>を参照すると良いかも(<a href="http://www.mozilla.org/docs/dom/" title="The DOM in Mozilla">The DOM in Mozilla</a>にZipped HTMLもあります)。Gecko専用という感じで、これに忠実に書いてもそのままではInternet Explorerで動かないとかやるせない感じですが、そこはほらアレ。</p>
