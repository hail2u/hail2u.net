Google Code PrettifyのSassハンドラ #2

<p><a href="https://github.com/hail2u/google-code-prettify-language-handlers">Missing Language Handlers for Google Code Prettify</a>のSass向けハンドラを更新して、Sass 3.3.0.rc.5に対応させた。<code>map-*()</code>や<code>str-*()</code>などの新しい関数を始め、<code>!optional</code>や<code>!global</code>といった新しいフラグを追加したり。他は特にいじっていない。</p>

<p>ついでに<a href="https://github.com/hail2u/google-code-prettify-language-handlers/releases/tag/v0.1.5">v0.1.5</a>ということにして、Bower化しておいた。</p>

<pre><samp>$ <kbd>bower install google-code-prettify-language-handlers</kbd>
</samp></pre>

<p>名前長いのでアレだけど、これでハンドラだけ入る。Google Code Prettify自体も<a href="https://github.com/tcollard/google-code-prettify">Bower化</a>されている(非公式にだけど)ので、導入はBowerで完結できる。</p>
