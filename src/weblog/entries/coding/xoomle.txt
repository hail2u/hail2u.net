XooMLe
<p><a href="http://www.dentedreality.com.au/xoomle/" title="XooMLe &#171; Projects &#171; Dented Reality">XooMLe</a>は、SOAPで提供されている<a href="http://www.google.com/apis/" title="Google Web APIs - Home">Google Web APIs</a>を、XML over HTTP(RESTで良いか)ラップするPHPスクリプトです。XSL書いたりするだけで、手軽に<a href="http://www.google.com/" title="Google">Google</a> Web APIsを扱える、つまり<a href="http://www.amazon.co.jp/exec/obidos/redirect?tag=hail2unet-22&amp;path=subst/associates/join/webservices.html" title="Amazon Webサービス">AWS</a>のRESTとほぼ同じ感じで扱えてしまう代物。</p>
<p>実際にどんな感じのものかは<a href="http://www.dentedreality.com.au/xoomle/search/" title="XooMLe">ブラウザ上でXMLを貰えるデモ</a>を参照すればわかるでしょう。実際にこのデモを使ってアプリを書いたら怒られます、多分。</p>
<p>そのうちGoogle側で対応しそうな気がしないでもないですが、それを促進する意味でも大きいかもしれない。</p>
