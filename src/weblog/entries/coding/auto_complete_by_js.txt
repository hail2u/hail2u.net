JavaScriptでオート・コンプリート
<p>普段からお世話になる機会が多い(と思う)、Internet Explorerのオート・コンプリート。<a href="http://www.webreference.com/programming/javascript/gr/column5/index.html" title="Creating a Textbox with JavaScript Auto-Complete - WebReference.com-">これに似たものをJavaScriptで作るという記事</a>が、<a href="http://www.webreference.com/" title="WebReference.com - The Webmaster&#39;s Reference Library - Web Authoring Tips &amp; Tutorials for Developers">WebReference.com</a>で公開されていました。</p>
<p>オート・コンプリート自体のコード的には一文字ずつ調べて候補を絞っていくというもので、シンプルなものですね。</p>
