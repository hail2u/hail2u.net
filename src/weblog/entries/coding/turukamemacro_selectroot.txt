SelectRoot
<p>SelectRootという鶴亀マクロを必要に駆られて作りました。鶴亀メールで実装されて無い機能で、もっとも必要だと勝手に考えているものに「未読メールの読み進めをループする」というものがあります。未読メールの読み進めはSpaceキーにてやってるわけですが、見た目的に下の方の最後の未読メールを読み終わると、見た目的に上の方に未読メールが残っていても、そこにジャンプしてくれません。これがまた辛い。ので、単純明快に解決してみました。</p>
<pre>loaddll &quot;tkinfo.dll&quot;;

if (dllfunc(&quot;IsTuruKame&quot;)) {
  #n = dllfunc(&quot;SelectFolder&quot;, &quot;一番上のアカウント名&quot;, &quot;&quot;);
}

endmacro;
</pre>
<p>このマクロを、マクロの登録→自動起動→受信が一段落した時に設定すると、受信が一段楽した時に、一番上にあるアカウントのルートを選択してくれるので、未読メールを一気に読み進めることができるようになります。「未読メールの読み進めをループする」わけではなく、「ループしなくても未読メールをすべて読み進めることができるようにする」というわけで。</p>
<p>激しく便利。とか、自画自賛してみる。</p>
<p>はるかに高機能なNextUnreadというマクロが<a href="http://hide.maruo.co.jp/lib/tkmacro/tkhlp129.html" title="秀まるおのホームページ－ライブラリ－鶴亀メールマクロライブラリ">鶴亀メールちょっと便利なマクロ集</a>に含まれているので、いわゆる「未読メールの読み進めをループする」機能が欲しい方はそちらを利用した方が良いと思います。</p>
