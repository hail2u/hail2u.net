File::Pathモジュール
<p>パスを指定して部分的に静的生成するblosxomのプラグインを作っていたのですが、存在しないディレクトリ以下にファイルを吐き出すときに、再帰的にディレクトリ作成をするコードを書くのがメンドウだったので、<a href="http://www.cpan.org/" title="CPAN">CPAN</a>漁ってみました。<a href="http://search.cpan.org/search?module=File::Path" title="search.cpan.org: File::Path - create or remove directory trees">File::Path</a>というモジュールにそういったサブルーチンがあるようで。</p>
<p>使い方は、</p>
<pre><code class="language-perl">my @dirs = mkpath(&#39;/home/nagashima/public_html/foo/bar&#39;);
</code></pre>
<p>だけと、とっても簡単でスバラシイ。<code>@dirs</code>には作成したディレクトリのパスが入るみたい。</p>
<pre><code class="language-perl">my $num = rmtree(&#39;/home/nagashima/public_html&#39;);
</code></pre>
<p>というような、ファイルとディレクトリを全て削除してくれるサブルーチンもあるようだ。怖くて使えないよ。</p>
<p>なかなかうまく探せなかったので、誰か日本語で逆引きCPANとか作ってください。機能からモジュール探せる奴。</p>
