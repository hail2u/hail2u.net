Opera Doubling Scrollbar Bug

Opera 11で二重にスクロールバーが出現するバグ。再現するの難しい。

html, body {
  margin: 0;
  padding: 0;
  height: 100%;
  overflow-x: hidden;
}

#footer .image {
  position: absolute;
  bottom: 0;
}

としてheight: 100%;で水平スクロールバーを消そうとした上に、何かの孫要素を下端に揃えるように配置しようとすると起こるようだ。
