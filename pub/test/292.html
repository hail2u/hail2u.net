SVG Feed Icon

背景色が完全に透明なSVGのフィード・アイコンです。SVGなのでキレイに拡大縮小でき、CSSを利用して好きなようにいじれます。

Demos

Default

背景色と角丸はCSSで行なっています。

Resize to 32x32 and 512x512

CSSのwidthプロパティを使ってリサイズしています。

Change background color to DodgerBlue

背景色が透明なので好きな色でアイコンが作成できます。

Add SVG Gradient

SVGグラデーションを重ねて凹凸感を出すのもCSSをいじるだけです。

Add CSS Gradient

CSSグラデーションならクロスブラウザ度は下がりますがもっと簡単でしょう。

Experimental

まん丸にしたり、枠線をつけたり、影をつけたり、傾けたり、表現力の増したCSSで色々できます。
