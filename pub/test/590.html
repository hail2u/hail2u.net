Spaces around Media Queries' "and" Keyword

メディア・クエリーのandキーワードの周囲には空白文字が必要だが、前の空白文字は省略できる。

()and()

If this test is green, the browser can parse ()and() media queries.

() and()

If this test is green, the browser can parse () and() media queries.

()and ()

If this test is green, the browser can parse ()and () media queries.
