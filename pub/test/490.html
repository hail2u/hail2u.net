Ellipsis and CSS text-* properties

Test Case

Nunc tempus magna sit amet metus hendrerit, pellentesque interdum tellus lobortis.

Summary

Ellipsis created by text-overflow does not respect CSS properties such as text-decoration, and there is no way to add CSS properties to ellipsis.
