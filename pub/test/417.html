HTML -> CSS -> SVG -> CSS

背景画像の二重引用符のSVGでもHTMLで参照しているCSSを参照してpath要素のfillプロパティーを#ff9900に指定している。

CSSで指定したSVGへのスタイルは反映されるか？

Chrome 19 (21)とSafari 5、Firefox 12 (14)では無指定とみなされて黒に、Internet Explorer 9 (10)とOpera 11 (12)ではCSSでの指定通り#ff9900になる。

仕様的にどちらが正しいかは未調査。

重複リクエストされるか？

Inernet Explorer 9ではCSSのリクエストが重複するバグがある(CSSがキャッシュ可能であれば片方は304 Not Modifiedにはなる)。

Internet Explorer 9の重複リクエストは開発者ツールでそう見えるだけで、Fiddlerで確認した限りでは実際にHTTPリクエストは飛ばなかった。Thanks to @mayuki
