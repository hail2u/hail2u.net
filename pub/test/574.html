Image Displacement on Hover

Chrome 35において、inline-blockやfloatである要素の子のブロック要素で%単位を使ってpaddingプロパティーを指定すると、オンマウスで画像の位置と余白が微妙に変化するバグ。アウトラインも初期状態ではおかしくなっている。

  • Internet Explorer 11: アウトラインはおかしいが、ずれは起こらない
  • Chrome 35: アウトラインがおかしく、オンマウスでずれる
  • Firefox 30: 問題なし
  • Opera 22: アウトラインがおかしく、オンマウスでずれる
  • Internet Explorer 12: アウトラインはおかしいが、ずれは起こらない
  • Chrome 37: アウトラインはおかしいが、ずれは起こらない
  • Firefox 32: 問題なし
  • Opera 23: アウトラインはおかしいが、ずれは起こらない

注: a要素にoutlineプロパティーを指定すると直る。
