:first-letterを利用したセレクタのIE 6でのバグ

A :first-letter擬似要素と{の間に半角スペースを入れれば問題なく反映される。

p.case-a:first-letter {
  color: red;
  font-size: 200%;
  font-weight: bold;
}

B :first-letter擬似要素と{の間に半角スペースを入れずに続けて書くと無視される。

p.case-b:first-letter{
  color: red;
  font-size: 200%;
  font-weight: bold;
}

Quirksモードではどちらの書き方でも反映される。
