WebKit/Blink Font Rendering Bug on Windows

→ Lorem Ipsum: Dolor Sit Amet

Webフォントとtext-rendering: optimizeLegibilityの組み合わせで一部のグリフが正常に表示されないことがある。WindowsのWebKit/Blinkで再現。

WindowsのChrome 28でのおかしくなった様子のスクリーンショット。矢印以外はすべてLimelightで表示されるべきだがそうならない。Webフォントに含まれないユニコード文字列などで異常が始まり、コロン等のASCII記号で正常に戻っている。環境依存の可能性もあるので要検証。
