with or without Generic Font Family

フォントを指定する時に汎用フォント・ファミリーを指定するかどうかにより、使われるフォントが変化する問題についてのテスト・ページです。Internet Explorer 11やEdge 14では汎用フォント・ファミリーを指定していない場合、メイリオの代わりにＭＳ Ｐゴシックが使われる。Firefox 50やChrome 54では汎用フォント・ファミリーがあってもなくても既定のフォント（標準フォント）に従う。

汎用フォント・ファミリーなし

X10 はホームオートメーションでの機器間通信に関する国際オープン工業規格である。屋内の電力配線を使って制御信号を送る（電力線搬送通信参照）。また、電波による通信も定義されている。

X10 は1975年、スコットランド Glenrothes にある Pico Electronics 社が家電製品の遠隔制御のために開発した。ホームオートメーション技術としては最も古くからあり、現在でも欧米を中心に利用されている。

より帯域幅の広い代替規格として、KNX、INSTEON、BACnet、LonWorks などが登場しているが、X10 は家庭用としては最も広く利用されている。

.without {
  font-family: Verdana;
}
