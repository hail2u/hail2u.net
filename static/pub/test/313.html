Opera CSS Hack for pointer-events

Fullscreen and click-through overlay is shown in all modern browsers excepting Opera.

.test {
  position: fixed;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  z-index: 1;
  background-color: black;
  opacity: 0.5;
  pointer-events: none;
}

x:-o-prefocus, .test {
  display: none;
}
