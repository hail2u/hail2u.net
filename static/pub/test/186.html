text-indent and transform

box

text-indentに大きな負の値を指定した要素を含む要素をtransformでマイナス方向に回転させるとスクロールバーが出てしまう。

      .box {
        width: 160px;
        height: 160px;
        line-height: 160px;
        color: #fff;
        background-color: #396;
        text-align: center;
        -webkit-transform: rotate(-30deg);
        box-shadow: inset 0 0 8px rgba(0, 0, 0, 0.5);
      }

      a {
        display: block;
        width: 160px;
        height: 160px;
        text-indent: -10000px;
      }
